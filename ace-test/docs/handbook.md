# ace-test Handbook Reference

## Skills

| Skill | What it does |
|-------|-------------|
| `as-test-plan` | Create task-level testing implementation plans |
| `as-test-create-cases` | Build test cases from behavior and coverage context |
| `as-test-fix` | Run structured test-failure repair workflow |
| `as-test-improve-coverage` | Detect coverage gaps and create tasks |
| `as-test-verify-suite` | Run suite health and quality checks |
| `as-test-optimize` | Profile and optimize slow tests |
| `as-test-performance-audit` | Audit test performance and produce actionable work |
| `as-test-review` | Review test work against quality standards |
