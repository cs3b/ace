# Goal 1 — Create Task

## Goal

Create a new draft task with `ace-task create` and capture the created task ID,
CLI output, and resulting task file content.

## Workspace

Save artifacts to `results/tc/01/`.

## Constraints

- Use `ace-task create` (with `--status draft`) to create the task.
- Capture stdout, stderr, and exit code.
- Record the new task reference in a file for reuse by later goals.
