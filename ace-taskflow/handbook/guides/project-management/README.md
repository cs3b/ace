### [Release Codenames Guide](release-codenames.g.md)

Comprehensive guide for selecting and applying codenames to software releases. Covers the philosophy, naming conventions, selection process, and integration with project workflows.

**Key Topics:**

- Codename philosophy and benefits
- Naming conventions and theme categories  
- Selection criteria and decision framework
- Implementation guidelines and validation
- Integration with draft-release workflow

This guide ensures consistency and meaning in release naming across the development lifecycle.
