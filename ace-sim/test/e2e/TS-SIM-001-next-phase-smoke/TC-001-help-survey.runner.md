# Goal 1: Help Survey

## Objective
Capture `ace-sim` help output and confirm `run` command plus preset-oriented options are visible.

## Steps
1. Run `ace-sim --help` and save stdout/stderr/exit to:
   - `results/tc/01/help.stdout`
   - `results/tc/01/help.stderr`
   - `results/tc/01/help.exit`
2. Run `ace-sim run --help` and save stdout/stderr/exit to:
   - `results/tc/01/run-help.stdout`
   - `results/tc/01/run-help.stderr`
   - `results/tc/01/run-help.exit`
