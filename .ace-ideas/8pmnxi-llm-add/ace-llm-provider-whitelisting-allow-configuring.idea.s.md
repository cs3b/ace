## Description

ace-llm provider whitelisting: Allow configuring which providers are active in the main llm config file. Two aspects: (1) provider configuration — not everyone needs all providers loaded, (2) provider whitelisting — explicit allow-list so only approved providers are available. This would let projects/users control which providers appear in ace-llm --list-providers without needing to manipulate individual provider config files or symlinks. Could be a simple whitelist/blacklist in .ace/llm/config.yml or similar.
