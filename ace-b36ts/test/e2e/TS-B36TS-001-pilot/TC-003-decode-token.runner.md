# Goal 3 — Decode Token

## Goal

Decode the token `i50jj3` using `ace-b36ts`. Save a file named `i50jj3` whose **content is the decoded result** (the date or timestamp the token represents).

## Workspace

Save the file to `results/tc/03/` as `i50jj3` with the decoded content.

## Constraints

- Use only `ace-b36ts` to perform the decoding. Do not compute the result manually.
- Using what you learned from Goal 1, decode a token. Do not assume syntax beyond what Goal 1 revealed.
- The file content must come from the tool's actual output, not fabricated.
- Do not use shell redirection on the `ace-b36ts` command itself.
- Decode the token and capture its output as `results/tc/03/i50jj3.stdout`, `results/tc/03/i50jj3.stderr`, `results/tc/03/i50jj3.exit`.
- Then save `results/tc/03/i50jj3` with the decode output from `results/tc/03/i50jj3.stdout` (for example using a plain copy command).
