# Task Manager List Alias

## Intention

To provide an alias `list` for the existing `all` command within the `task-manager` tool to simplify user interaction and offer a more intuitive command for viewing all tasks.

## Problem It Solves

**Observed Issues:**
- Users might expect a command named `list` to view all tasks, as this is a common convention in many CLI tools.
- The current `task-manager all` command, while functional, might not be immediately intuitive for all users.
- Lack of a `list` command creates a minor inconsistency with common CLI patterns.

**Impact:**
- Users may have to consult documentation or guess the correct command to view all tasks.
- Slightly reduced user experience due to a missing common command alias.
