# Add OpenRouter LLM Provider

## Scope

Integrate OpenRouter as a new LLM provider in ace-llm gem. OpenRouter provides unified access to 400+ models from multiple providers through a single OpenAI-compatible API.

## API Details

| Property | Value |
|----------|-------|
| Base URL | `https://openrouter.ai/api/v1` |
| Endpoint | `/api/v1/chat/completions` |
| Auth | `Authorization: Bearer $OPENROUTER_API_KEY` |
| Compatibility | OpenAI-compatible |
| Extra Headers | `HTTP-Referer`, `X-Title` (optional, for app attribution) |

## Models

Model names use `provider/model` format:
- `openai/gpt-4o`
- `anthropic/claude-3.5-sonnet`
- `google/gemini-2.0-flash`
- `meta-llama/llama-3.1-70b-instruct`
