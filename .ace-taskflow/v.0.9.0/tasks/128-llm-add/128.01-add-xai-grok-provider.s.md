# Add x.ai (Grok) LLM Provider

## Scope

Integrate x.ai (Grok) as a new LLM provider in ace-llm gem. The x.ai API is OpenAI-compatible, making implementation straightforward.

## API Details

| Property | Value |
|----------|-------|
| Base URL | `https://api.x.ai/v1` |
| Endpoint | `/v1/chat/completions` |
| Auth | `Authorization: Bearer $XAI_API_KEY` |
| Compatibility | OpenAI-compatible |

## Models

- `grok-4` (latest)
- `grok-3` (default)
- `grok-3-fast`
- `grok-3-mini`
- `grok-2`
