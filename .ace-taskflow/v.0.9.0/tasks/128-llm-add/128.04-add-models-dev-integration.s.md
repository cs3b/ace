# Add models.dev Integration (ace-llm-models-dev)

## Scope

Create a new gem `ace-llm-models-dev` that integrates with [models.dev](https://models.dev/) - an open-source database of AI models. This enables model validation, cost tracking, and change monitoring across providers.

## Data Source

| Property | Value |
|----------|-------|
| API | `https://models.dev/api.json` |
| Source | [github.com/sst/models.dev](https://github.com/sst/models.dev) |
| Format | JSON (generated from TOML) |
| Logos | `https://models.dev/logos/{provider}.svg` |

## Capabilities

### 1. Model Validation
- Validate model names exist for a provider
- Check if model is deprecated/alpha/beta
- Local cache avoids repeated API calls

### 2. Cost Tracking
- Query cost per million tokens (input/output/reasoning/cache)
- Calculate query costs based on token usage
- Support audio pricing where available

### 3. Change Monitoring
- Download and cache API snapshots
- Generate diff between updates
- Track new models, deprecations, pricing changes

## Model Data Structure

From models.dev API, each model includes:

```yaml
# Core
name: "Model Display Name"
model_id: "provider:model-name"  # AI SDK format

# Capabilities
attachment: true/false
reasoning: true/false
tool_call: true/false
structured_output: true/false
temperature: true/false

# Dates
knowledge: "2024-04"
release_date: "2025-02-19"
updated_date: "2025-03-01"

# Cost (per million tokens)
cost:
  input: 3.00
  output: 15.00
  reasoning: 12.00  # optional
  cache_write: 3.75  # optional
  cache_read: 0.30   # optional

# Limits
limit:
  context: 400000
  output: 8192

# Modalities
modalities:
  input: [text, image, audio, video, pdf]
  output: [text, audio]

# Status
status: deprecated/alpha/beta  # optional
open_weights: true/false
```

## CLI Examples

```bash
# Sync local cache
ace-llm-models sync
# Synced 847 models from 42 providers

# Validate model name
ace-llm-models validate xai:grok-3
# ✓ xai:grok-3 is valid

ace-llm-models validate xai:grok-99
# ✗ xai:grok-99 not found. Did you mean: grok-3, grok-4?

# Calculate cost
ace-llm-models cost openai:gpt-4o --input 10000 --output 2000
# Input: 10K tokens × $2.50/M = $0.025
# Output: 2K tokens × $10.00/M = $0.020
# Total: $0.045

# Show diff since last sync
ace-llm-models diff
# + anthropic:claude-4-opus (new)
# ~ openai:gpt-4o: output cost $10.00 → $7.50
# - google:gemini-1.0-pro (deprecated)
```

## Integration with ace-llm

Optional integration for query validation:

```ruby
# In ace-llm configuration (.ace/llm/config.yml)
validate_models: true  # Enable model validation before queries
cache_path: ~/.cache/ace-llm-models-dev  # Local cache location
```
