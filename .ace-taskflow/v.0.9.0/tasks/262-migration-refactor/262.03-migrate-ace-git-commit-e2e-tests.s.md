**Total: 7 scenarios, 20 TCs, ~35 fixture files**

## Technical Approach

### Setup Pattern

All scenarios follow:
```yaml
setup:
  - git-init
  - copy-fixtures
  - run: git add -A && git commit -m "initial" --quiet
  - env:
      PROJECT_ROOT_PATH: .
```

The 004a-d variants additionally commit package structure before test modifications.

### Variant Naming

Preserve `004a/b/c/d` suffixes in directory names:
- `MT-COMMIT-004a-*.mt.md` → `TS-COMMIT-004a-*/`

### TC Extraction

- Strip `ace-test-e2e-sh "$TEST_DIR"` wrappers
- Preserve `$PROJECT_ROOT/bin/ace-git-commit` references as just `ace-git-commit`
- Many TCs modify files then run commit — keep as TC steps
