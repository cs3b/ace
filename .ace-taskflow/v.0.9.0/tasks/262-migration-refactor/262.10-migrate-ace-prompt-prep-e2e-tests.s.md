# Migrate ace-prompt-prep E2E Tests

## Scope

Convert 1 `.mt.md` file (5 TCs) in `ace-prompt-prep/test/e2e/` to the new directory-based per-TC format. All test cases are independent. Smallest migration task.

## Migration Inventory

| Source File | Target Directory | TCs | Key Fixtures |
|-------------|-----------------|-----|-------------|
| `MT-PREP-001-basic-prompt-processing.mt.md` | `TS-PREP-001-basic-prompt-processing/` | 5 | sample-prompt.md (with bundle frontmatter), minimal-prompt.md (no frontmatter), task-prompt.md (with task ID) |

**Total: 1 scenario, 5 TCs, 3 fixtures**

## Technical Approach

### Setup Pattern

```yaml
setup:
  - git-init
  - copy-fixtures
  - env:
      PROJECT_ROOT_PATH: .
```

### Key Characteristics

- Tests ace-prompt-prep workspace setup and archival
- Uses `.cache/ace-prompt-prep/prompts/` workspace directory
- Tests symlink management (the-prompt.md)
- Integration with ace-bundle and ace-timestamp

## Implementation Plan

### Execution Steps

- [ ] Create `TS-PREP-001-basic-prompt-processing/` directory
- [ ] Extract 3 fixture files (sample-prompt.md, minimal-prompt.md, task-prompt.md)
- [ ] Extract `scenario.yml` with git-init, copy-fixtures, env
- [ ] Extract 5 TC files (TC-001 through TC-005)
  > TEST: Setup Validation

- [ ] Remove original `.mt.md` file

## Acceptance Criteria

- [ ] Scenario setup succeeds
