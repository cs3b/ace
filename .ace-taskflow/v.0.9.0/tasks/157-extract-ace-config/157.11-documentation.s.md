### 1. README.md for ace-config

```markdown
# ace-config

Generic configuration cascade management for Ruby applications.

## Installation

gem 'ace-config'

## Usage

### Basic Usage
config = Ace::Config.new
config.resolve_for(["settings.yml"])

### Custom Folder Names
config = Ace::Config.new(
  config_dir: ".my-app",
  defaults_dir: ".my-app-defaults"
)

### With Gem Defaults
config = Ace::Config.new(
  gem_path: File.expand_path("../..", __dir__)
)

## Cascade Order
1. $CWD/{config_dir}/
2. Intermediate directories
3. $PROJECT_ROOT/{config_dir}/
4. $HOME/{config_dir}/
5. $GEM_PATH/{defaults_dir}/

## Configuration Options
- config_dir: User config folder name (default: ".ace")
- defaults_dir: Gem defaults folder name (default: ".ace-defaults")
- gem_path: Path to gem root for defaults
- project_markers: Array of files to detect project root
- merge_strategy: How to merge arrays (:replace, :concat, :union)
```

```markdown
# Migrating to ace-config

## For ace-* gem developers
1. Replace Ace::Core config calls with Ace::Config
2. Pass gem_path for defaults
3. Use .ace-defaults/ instead of .ace.example/

## For external projects
1. Add gem 'ace-config' to Gemfile
2. Configure folder names as needed
3. No ace-* dependencies required
```
