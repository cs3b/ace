# Rename .ace.example/ to .ace-defaults/ + Cascade Integration

## Objective

1. Rename `.ace.example/` to `.ace-defaults/` across all ace-* gems
2. Integrate gem defaults into ConfigFinder cascade with explicit gem path passing

## Architecture Decision

### New Cascade Order (highest to lowest priority)

```
1. $CWD/.ace/                    # Current working directory
2. ... intermediate dirs ...     # All .ace/ dirs between CWD and PROJECT_ROOT
3. $PROJECT_ROOT/.ace/           # Project root (auto-discovered via markers)
4. $HOME/.ace/                   # User home preferences
5. $GEM_PATH/.ace-defaults/      # Gem defaults (LOWEST - fallback only)
```

**Note:** DirectoryTraverser already handles #1-#4 by walking from CWD to PROJECT_ROOT.
The change is adding #5 (gem defaults) at the end of the cascade.

### New API

```ruby
# Gems pass their path when requesting config
gem_root = Gem.loaded_specs["ace-git"]&.gem_dir || File.expand_path("../..", __dir__)
resolver = Ace::Core::Organisms::ConfigResolver.new(gem_path: gem_root)
config = resolver.resolve_for(["git/config.yml"]).data

# Or via simplified API
Ace::Core.get("git", file: "config", gem_path: gem_root)
```
