# Build the ace-sim Ruby Gem

## Objective

Wrap the PROVEN workflows from Subtask 296.01 into a proper ace-sim gem. Does NOT redesign them — implements the exact contracts validated in Phase 1.

## Behavioral Specification

### User Experience
- **Input**: User runs `ace-sim run --config <name> --source <file>`
- **Process**: Gem reads YAML config, executes step pipeline via ace-bundle + ace-llm subprocesses, caches all artifacts
- **Output**: Step artifacts + synthesis report in `.cache/ace-sim/simulations/<run-id>/`

### Expected Behavior

```
$ ace-sim run --config review-next-phase --source my-idea.s.md

ace-sim v0.1.0 | run-id: 8pq4ab

[1/2] draft (google:flash)
  system: ace-bundle sim/step-draft-context (1247 lines)
  source: my-idea.s.md (42 lines)
  ... done (4.2s, 2847 tokens)
  => .cache/ace-sim/simulations/8pq4ab/step-01-draft/output.md

[2/2] plan (google:flash)
  input: step-01-draft/output.md + source
  ... done (3.8s, 2103 tokens)
  => .cache/ace-sim/simulations/8pq4ab/step-02-plan/output.md

[synthesis] (google:flash) ... done (2.1s)
  => .cache/ace-sim/simulations/8pq4ab/synthesis/report.md

Questions: 7 | Refinements: 4
```

### Interface Contract

```bash
# Full simulation run
ace-sim run --config review-next-phase --source my-idea.s.md

# Dry run (show plan without LLM calls)
ace-sim run --config review-next-phase --source input.md --dry-run

# Single step with specific provider
ace-sim run --config review-next-phase --source input.md --step draft --provider google:flash

# List available simulation configs
ace-sim list

# Show run summary
ace-sim show <run-id>
```

**Error Handling:**
- Missing config: `Error: config 'foo' not found. Available: review-next-phase (ace-sim list)`
- Missing source: `Error: source file not found: path/to/missing.md`
- Provider failure: `Error: ace-llm failed for step 'draft' (google:flash): [error message]`
- Partial failure: Previous step artifacts preserved, synthesis skipped

### Verification Evidence

- **Command(s)**: `ace-sim run --config review-next-phase --source <real-idea-file>`
- **Artifacts**:
  - `.cache/ace-sim/simulations/<run-id>/step-01-draft/output.md`
  - `.cache/ace-sim/simulations/<run-id>/step-02-plan/output.md`
  - `.cache/ace-sim/simulations/<run-id>/synthesis/report.md`
- **Expected vs Observed**: pass when all artifacts exist with non-empty content and step chaining is verified
- **Failure Classification**: `implementation failure | provider unavailable | environment issue`

### Success Criteria

- [ ] `ace-sim run --config review-next-phase --source <file>` works end-to-end
- [ ] `ace-sim run --dry-run` shows plan without LLM calls
- [ ] `ace-sim list` shows available configs
