# Add pr: support to ace-context

## Objective

Add `pr:` as a new source type in ace-context, enabling PR diff content to be loaded alongside files, diffs, and commands.

## Behavioral Specification

### User Experience

```yaml
# ace-context configuration with pr: support
files: ["lib/**/*.rb"]
diffs: ["origin/main..HEAD"]
pr: 123                                    # NEW - PR number
pr: "owner/repo#456"                       # Cross-repo PR
pr: "https://github.com/owner/repo/pull/789"  # Full URL
```

### Interface Contract

```bash
# Load PR diff via ace-context
ace-context -f config.yml  # where config.yml has pr: 123

# Inline via ace-review (after 145.02)
ace-review --subject pr:123
```

### Expected Output

PR diff content included in ace-context output, similar to how diffs are included.

## Success Criteria

- [ ] `pr:` key recognized in ace-context config schema
- [ ] PR number format: `pr: 123`
- [ ] Qualified format: `pr: "owner/repo#456"`
- [ ] URL format: `pr: "https://github.com/owner/repo/pull/789"`
- [ ] Integration with `gh` CLI for PR diff fetching
- [ ] Error handling: gh not installed, PR not found, auth issues
- [ ] Tests cover all PR identifier formats

## Implementation Notes

### Affected Components

1. **Config Schema** - Add `pr:` as valid key
2. **New Atom: PrExtractor** (`lib/ace/context/atoms/pr_extractor.rb`)
   - Parse PR identifier formats
   - Execute `gh pr diff` safely
   - Handle errors gracefully
3. **ContextLoader** - Integrate PR extraction alongside files/diffs/commands
4. **Tests** - Unit tests for PR parsing, integration tests with mocked gh CLI

### PR Identifier Parsing

```ruby
module PrExtractor
  def self.parse_identifier(input)
    case input.to_s
    when /^(\d+)$/                           # Simple number
      { number: $1, repo: nil }
    when /^([^\/]+\/[^#]+)#(\d+)$/           # owner/repo#123
      { number: $2, repo: $1 }
    when /github\.com\/([^\/]+\/[^\/]+)\/pull\/(\d+)/  # Full URL
      { number: $2, repo: $1 }
    else
      raise ArgumentError, "Invalid PR identifier: #{input}"
    end
  end

  def self.fetch_diff(identifier)
    parsed = parse_identifier(identifier)
    args = ["gh", "pr", "diff", parsed[:number]]
    args += ["-R", parsed[:repo]] if parsed[:repo]

    stdout, stderr, status = Open3.capture3(*args)
    raise "gh pr diff failed: #{stderr}" unless status.success?
    stdout
  end
end
```

### Error Messages

```
Error: GitHub CLI (gh) not installed. Install with: brew install gh
Error: PR #123 not found in repository
Error: Not authenticated with GitHub. Run: gh auth login
```

## Out of Scope

- PR metadata (title, author, state) - just the diff
- PR comments - handled separately by ace-review
- Posting comments - ace-review responsibility
