# Feedback Model and Core Atoms

## Behavioral Specification

### User Experience
- **Input**: Raw feedback data (title, files, reviewer, priority, finding content)
- **Process**: System creates FeedbackItem with unique 6-char ID and URL-safe slug
- **Output**: FeedbackItem object ready for serialization to file

### Expected Behavior

The FeedbackItem model holds all data for a single review finding. Each item gets a unique 6-character ID (via ace-support-timestamp Base36 encoding) and a slug derived from its title for human-readable filenames.

The model should:

1. **Store feedback attributes**: id, title, files, reviewer, status, priority, created/updated timestamps, sections (finding, context, research, resolution)
2. **Generate unique IDs**: Use ace-support-timestamp for compact, sortable 6-char IDs
3. **Generate URL-safe slugs**: Convert titles to lowercase, hyphenated slugs (max 40 chars)
4. **Track status transitions**: draft -> pending/invalid/skip -> done
5. **Validate required fields**: id, title, status must be present

### Interface Contract

```ruby
# FeedbackItem model
class FeedbackItem
  attr_accessor :id, :title, :files, :reviewer, :status, :priority,
                :created, :updated, :finding, :context, :research, :resolution

  VALID_STATUSES = %w[draft pending invalid skip done].freeze
  VALID_PRIORITIES = %w[critical high medium low info].freeze

  def initialize(attrs = {})
  def valid?
  def filename  # => "8o7abc-security-sql-injection.s.md"
  def to_frontmatter  # => YAML hash for file header
  def to_markdown  # => full file content with frontmatter and sections
  def archived?  # => status in [invalid, skip, done]
end

# FeedbackIdGenerator atom
class FeedbackIdGenerator
  def self.generate  # => "8o7abc" (6-char Base36)
end

# FeedbackSlugGenerator atom
class FeedbackSlugGenerator
  def self.generate(title, max_length: 40)  # => "security-sql-injection"
end
```

**Error Handling:**
- Missing required field: Raise ArgumentError with field name
- Invalid status: Raise ArgumentError with valid options
- Invalid priority: Raise ArgumentError with valid options

**Edge Cases:**
- Empty title: Raise validation error
- Very long title: Truncate slug at word boundary
- Special characters in title: Replace with hyphens, collapse multiple hyphens
