# Synthesizer Integration

## Behavioral Specification

### User Experience
- **Input**: Review completes with `--feedback` (default) or `--no-feedback` flag
- **Process**: After synthesis, system extracts feedback items if enabled
- **Output**: Feedback files created alongside synthesis report

### Expected Behavior

Integrate FeedbackManager with the existing review workflow:

1. **Default behavior**: Reviews extract feedback after synthesis
2. **Opt-out**: `--no-feedback` flag disables extraction
3. **Backward compatible**: Synthesis report still generated as before
4. **Additive**: Feedback files complement, don't replace synthesis

The integration points are:
- ReportSynthesizer calls FeedbackManager.extract_and_save after generating synthesis
- ReviewManager passes feedback option through the chain
- ReviewOptions model gains `feedback` boolean option

### Interface Contract

```ruby
# Updated ReviewOptions
class ReviewOptions
  attr_accessor :feedback  # default: true
end

# Updated ReportSynthesizer
class ReportSynthesizer
  def initialize(..., feedback_manager: nil)

  def synthesize(reports, options)
    # Existing synthesis logic
    result = generate_synthesis(reports)

    # New: Extract feedback if enabled
    if options.feedback && @feedback_manager
      feedback_result = @feedback_manager.extract_and_save(reports)
      result[:feedback] = feedback_result
    end

    result
  end
end

# Updated ReviewManager
class ReviewManager
  def execute_multi_model(options)
    # ... existing code ...

    # Pass feedback option to synthesizer
    synthesis_result = synthesizer.synthesize(
      model_results,
      feedback: options.feedback
    )
  end
end
```

**CLI Changes:**
```bash
# Default (feedback enabled)
ace-review --preset code-pr

# Feedback disabled
ace-review --preset code-pr --no-feedback
```

**Error Handling:**
- Extraction failure: Log warning, don't fail overall review
- No FeedbackManager: Skip extraction silently (backward compat)

**Edge Cases:**
- Single-model review: Still extracts feedback from single report
- Empty synthesis: No findings to extract, return empty
