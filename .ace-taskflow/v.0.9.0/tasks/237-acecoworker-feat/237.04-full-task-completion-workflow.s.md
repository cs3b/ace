# Full Task-Completion Workflow

## Overview

Production workflow with variable interpolation at prepare time. Simplified from 234.05.

## Objective

Enable variable interpolation when preparing sessions:
- `{{taskref}}`, `{{branch}}` placeholders in presets
- `ace-coworker prepare --preset X --var taskref=148`
- Variables resolved at prepare time (not runtime)
- Enhance existing presets with variable placeholders

## Scope of Work

### Simplified from 234.05

**Included:**
- Variable interpolation in preset YAML at prepare time
- `--var key=value` flag on prepare command
- Standard variables: `{{taskref}}`, `{{branch}}`, `{{date}}`
- Updated presets with variable placeholders

**Excluded (from original 234.05):**
- Runtime variable interpolation
- Complex 8-step workflow definition
- Human gates (moved to 240.01)
- Variable extraction from text
- $var syntax (use {{var}} for clarity)

## Technical Approach

### Variable Syntax

Use Mustache-style `{{variable}}` for clarity:

```yaml
# preset with variables
name: work-on-task
description: Work on task {{taskref}}

steps:
  - name: implement
    instructions: |
      Execute task {{taskref}} implementation.
      Branch: {{branch}}
      Report with: ace-coworker report <file>
```

### CLI Interface

```bash
# Prepare with variables
ace-coworker prepare --preset work-on-task \
  --var taskref=148 \
  --var branch=148-feature

# Variables can also come from environment
export ACE_TASKREF=148
ace-coworker prepare --preset work-on-task
```

### VariableInterpolator Atom

```ruby
module Ace::Coworker::Atoms
  class VariableInterpolator
    VARIABLE_PATTERN = /\{\{([a-z_][a-z0-9_]*)\}\}/i

    # Pure function: interpolate variables in text
    # @param text [String] Text with {{variable}} placeholders
    # @param variables [Hash] Variable values { taskref: "148" }
    # @return [String] Text with variables replaced
    def self.interpolate(text:, variables:)
      text.gsub(VARIABLE_PATTERN) do |match|
        var_name = Regexp.last_match(1).downcase.to_sym
        variables.fetch(var_name, match) # Keep original if not found
      end
    end

    # Pure function: extract variable names from text
    # @param text [String] Text with {{variable}} placeholders
    # @return [Array<Symbol>] List of variable names
    def self.extract_variables(text:)
      text.scan(VARIABLE_PATTERN).flatten.map { |v| v.downcase.to_sym }.uniq
    end
  end
end
```

### Standard Variables

| Variable | Source | Description |
|----------|--------|-------------|
| `{{taskref}}` | --var or env | Task reference (e.g., "148") |
| `{{branch}}` | --var or git | Current git branch |
| `{{date}}` | auto | Current date (YYYY-MM-DD) |
| `{{session}}` | auto | Session ID |

### Updated Preset Example

```yaml
# work-on-task.yml
name: work-on-task
description: Standard task completion workflow for {{taskref}}
variables:
  - taskref: required
  - branch: optional

steps:
  - name: implement
    instructions: |
      Execute task {{taskref}} implementation following the task spec.

      Reference: ace-taskflow task {{taskref}}
      Branch: {{branch}}

      When complete, report with: ace-coworker report <file>

  - name: test
    instructions: |
      Run tests to verify implementation:
      ace-test

      Report results with: ace-coworker report <file>
    verification: ace-test

  - name: commit
    instructions: |
      Commit the changes:
      ace-bundle wfi://commit

      Report with: ace-coworker report <file>
```
