# 226.05: Progressive Strategy (Context Carryover)

## Goal

Sequential review with findings summary carried forward between chunks.

## Deliverables

1. `ace-review/lib/ace/review/molecules/strategies/progressive_strategy.rb`
2. Update synthesis to handle progressive flow
3. Unit tests

## Flow

```
Chunk 1 → Review → Extract findings summary
    ↓
Chunk 2 + "Previous findings: ..." → Review → Extract findings
    ↓
Chunk 3 + "Previous findings: ..." → Review
    ↓
Final synthesis with all findings
```

## Interface

```ruby
strategy = SubjectStrategy.for(:progressive, {
  max_tokens_per_chunk: 100_000,
  summary_prompt: "Summarize your findings in 2-3 sentences for context carryover."
})

# Progressive returns units with carryover instructions
units = strategy.prepare(subject, context)
# → [
#     { content: "...", metadata: { chunk_index: 0, requires_summary: true } },
#     { content: "...", metadata: { chunk_index: 1, inject_previous_summary: true } },
#     ...
#   ]
```

## Execution

The `MultiModelExecutor` must handle progressive strategy specially:
1. Execute chunk 0
2. Extract summary from response
3. Inject summary into chunk 1 prompt
4. Execute chunk 1
5. Extract summary, inject into chunk 2...
6. Final synthesis

## Acceptance Criteria

- [ ] Each chunk review receives summary of prior findings
- [ ] Sequential execution (not parallel)
- [ ] Summary extraction works with standard review output
- [ ] Final report shows progressive context was used
- [ ] Falls back to chunked if summary extraction fails
- [ ] All tests pass
