# Create documentation workflow template

## Objective

Create a recipe and preset for ace-assign that guides documentation updates from gap identification through commit.

## Scope

Create documentation recipe and preset with:
1. **Onboard** - Load project context
2. **Research** - Find documentation gaps (outdated, missing, incorrect)
3. **Write** - Create or update documentation content
4. **Commit** - Create atomic commit with docs changes

## Deliverables

- [x] `catalog/recipes/documentation.recipe.yml` - Recipe definition
- [x] `presets/work-on-docs.yml` - Preset for direct `/ace:assign-prepare` use

- [x] Recipe loads with `ace-nav wfi://*` discovery
