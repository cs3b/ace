# Gem scaffold, models, and atoms

## Objective

Create the ace-overseer gem with standard structure, configuration cascade, data models, and pure-function atoms. After this subtask, `ace-overseer version` works and all atoms have passing tests.

## Behavioral Specification

### User Experience

```bash
ace-overseer version
# ace-overseer 0.1.0

ace-overseer --help
# Commands:
#   ace-overseer work-on    Create worktree, open tmux window, prepare assignment
#   ace-overseer status     Show status of all active task worktrees
#   ace-overseer prune      Clean up completed task worktrees
#   ace-overseer version    Print version
```

### Scope of Work

**Gem structure** ÔÇö follow `ace-tmux` as template:
- `ace-overseer/lib/ace/overseer.rb` ÔÇö module with config cascade (ADR-022)
- `ace-overseer/lib/ace/overseer/version.rb` ÔÇö VERSION = "0.1.0"
- `ace-overseer/lib/ace/overseer/cli.rb` ÔÇö dry-cli registry with command stubs
- `ace-overseer/exe/ace-overseer` ÔÇö executable wrapper
- `ace-overseer/ace-overseer.gemspec` ÔÇö dependencies on sibling gems
- `ace-overseer/.ace-defaults/overseer/config.yml` ÔÇö defaults
- `ace-overseer/Rakefile`, `CHANGELOG.md`, `README.md`, `LICENSE`
- `ace-overseer/test/test_helper.rb`
- `ace-overseer/handbook/{agents,workflow-instructions}/` ÔÇö empty directories
- Add to root `Gemfile`

**Config defaults** (`.ace-defaults/overseer/config.yml`):
```yaml
tmux_session_name: "ace"
window_preset: "cc"
default_assign_preset: "work-on-task"
window_name_format: "t{task_id}"
```

**Models:**
- `WorkContext` ÔÇö task_id, worktree_path, branch, assignment_status (Hash|nil), git_status (Hash|nil), tmux_window (String|nil)
- `PruneCandidate` ÔÇö task_id, worktree_path, assignment_complete (bool), task_done (bool), git_clean (bool), reasons (Array<String>)

**Atoms (pure functions):**
- `WindowNameFormatter.format(task_id, format:)` Ôćĺ e.g., "t230"
- `PresetResolver.resolve(task_frontmatter:, cli_preset:, default:)` Ôćĺ preset name
- `StatusFormatter.format_dashboard(contexts, release_info)` Ôćĺ terminal string
- `StatusFormatter.format_row(context)` Ôćĺ single row

### Success Criteria

- [ ] `ace-overseer version` outputs version string
- [ ] `ace-overseer --help` lists all commands (stubs OK for work-on, status, prune)
- [ ] Config cascade loads defaults from `.ace-defaults/overseer/config.yml`
- [ ] `ace-test ace-overseer` passes ÔÇö atoms and models have unit tests
- [ ] Gem is loadable: `require "ace/overseer"` works
- [ ] Root `bundle install` succeeds with new gem
