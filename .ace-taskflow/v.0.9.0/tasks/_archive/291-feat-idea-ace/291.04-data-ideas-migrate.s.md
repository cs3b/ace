# Migrate Ideas Data to .ace-ideas/ (One-Time Codemod)

## Objective

Move all existing idea files from `.ace-taskflow/v.0.9.0/ideas/` to the new
standalone `.ace-ideas/` root directory, making them discoverable by `ace-idea list`,
`ace-idea show`, and other ace-idea commands.

This is a one-time data migration. The migration logic lives in `codemods/` within
this task folder — not as a permanent gem command.

## Behavioral Specification

### User Experience
- **Input**: Existing ideas at `.ace-taskflow/v.0.9.0/ideas/` (137 `.idea.s.md` files)
- **Process**: Run `ruby codemods/migrate_ideas.rb --dry-run` to preview, then without
  `--dry-run` to execute
- **Output**: All ideas accessible via `ace-idea list`, structure preserved, no data loss

### Expected Behavior

1. Active ideas (top-level b36ts folders) move to `.ace-ideas/` root
2. `_archive/` moves intact to `.ace-ideas/_archive/`
3. `_maybe/` moves intact to `.ace-ideas/_maybe/`
4. `maybe/` (no underscore) is normalized: its contents move into `.ace-ideas/_maybe/`
5. The `2025111-ace-packages-review/` group folder moves intact (scanner recurses into it)
6. Legacy-named archive items (no b36ts ID) are moved silently — they won't appear in
   `ace-idea list` but are preserved for reference
7. `--dry-run` prints each action without writing

### Interface Contract

```bash
# Preview (no writes)
ruby .ace-taskflow/v.0.9.0/tasks/_archive/291-feat-idea-ace/codemods/migrate_ideas.rb \
  --dry-run

# Execute
ruby .ace-taskflow/v.0.9.0/tasks/_archive/291-feat-idea-ace/codemods/migrate_ideas.rb

# Verify
ace-idea list                  # ~23 active ideas
ace-idea list --in archive     # ~97 ideas
ace-idea list --in maybe       # ~9 ideas
ace-idea show 8pq0f8           # spot-check active idea
ace-idea show 8poyif           # confirm migrated orphan is in _maybe
```

### Success Criteria

- [ ] All b36ts ideas appear in `ace-idea list` / `ace-idea list --in {folder}`
- [ ] `ace-idea show <id>` works for sampled ideas from each folder
- [ ] `maybe/` is gone; its contents are in `_maybe/`
- [ ] `.ace-taskflow/v.0.9.0/ideas/` directory is empty (or removed)
- [ ] No data loss — all files preserved with original content
- [ ] `--dry-run` prints actions without modifying filesystem

## Scope of Work

- Ruby codemod script at `codemods/migrate_ideas.rb` (run once, then keep for audit trail)
- Script handles: move directories, normalize `maybe/` → `_maybe/`, skip legacy-named items
  with a printed warning

## Out of Scope

- Permanent `ace-idea migrate` command (overkill for a one-time migration)
- Migrating tasks or retros (covered by separate tasks 290/292)
