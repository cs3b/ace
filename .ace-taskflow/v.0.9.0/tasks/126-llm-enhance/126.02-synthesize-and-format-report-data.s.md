# Report Synthesis

## Scope

Combine multiple model reviews into a single consolidated report with LLM-powered analysis identifying consensus, conflicts, and prioritized actions.

## User Experience

```bash
# Standalone synthesis command
ace-review synthesize --session .cache/ace-review/sessions/review-20251201/

# With custom synthesis model
ace-review synthesize --session ./session-dir --synthesis-model gpt-4

# Auto-triggered after multi-model execution (when 2+ models complete)
ace-review --preset pr --model "gemini,gpt-4" --auto-execute
# → Automatically runs synthesis after individual reviews complete
```

## Expected Output

```

  Reading: review-gemini-2.5-flash.md (2.4 KB)
  Reading: review-gpt-4.md (3.1 KB)
  Reading: review-claude-sonnet.md (2.8 KB)

Generating synthesis with gpt-4...
✓ Synthesis complete

Saved: .cache/ace-review/sessions/review-20251201/synthesis-report.md

Summary:
  Common findings: 5 (high confidence)
  Unique insights: 3
  Conflicts resolved: 1
  Action items: 8
```

## Synthesis Report Format

```markdown
# Multi-Model Review Synthesis

## Overview
- Models: gemini-2.5-flash, gpt-4, claude-sonnet
- Subject: PR #50 (origin/main...feature-branch)
- Generated: 2025-12-01 17:55:00

## Consensus Findings (All Models Agree)
1. [High confidence finding]
2. ...

## Strong Recommendations (2+ Models)
1. [Recommendation with model attribution]

## Unique Insights
### From gemini-2.5-flash:
- [Unique observation]

## Conflicting Views & Resolution
### Topic: [e.g., Error handling]
- **gemini**: [Opinion A]
- **gpt-4**: [Opinion B]
- **Suggested resolution**: [Synthesis]

## Prioritized Action Items
1. [High priority - consensus]
2. [Medium priority]

## Individual Report References
- [gemini-2.5-flash](./review-gemini-2.5-flash.md)
```
