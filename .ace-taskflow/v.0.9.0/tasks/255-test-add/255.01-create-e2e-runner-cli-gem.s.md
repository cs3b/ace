# Create ace-test-e2e-runner-cli Gem Structure

## Objective

Create the gem skeleton for `ace-test-e2e-runner-cli` following ACE gem patterns with ATOM architecture.

## Gem Relationship Note

This gem (`ace-test-e2e-runner-cli`) is the **CLI tool** for running E2E tests. It is distinct from:

- **`ace-test-e2e-runner`** (existing): Contains E2E test infrastructure, templates, and the `*.mt.md` test format specification. Used by agents to define and organize tests.
- **`ace-test-e2e-runner-cli`** (this gem): Provides the `ace-e2e-test` CLI that discovers and executes tests via LLM.

The CLI gem depends on the infrastructure gem for test discovery patterns and format parsing.
