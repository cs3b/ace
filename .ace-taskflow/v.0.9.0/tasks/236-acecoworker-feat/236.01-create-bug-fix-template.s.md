# Create bug-fix workflow template

## Objective

Create a multi-step workflow template for ace-coworker that guides bug fixing from analysis through PR creation.

## Scope

Create `.ace-defaults/coworker/workflows/bug-fix.wf.yml` with sequential steps:

1. **Analyze** - Read bug report, understand expected vs actual behavior
2. **Reproduce** - Create minimal reproduction, verify bug exists
3. **Fix** - Implement the fix following project patterns
4. **Test** - Add/update tests to cover the bug case
5. **Commit** - Create atomic commit with conventional message
6. **PR** - Create pull request with bug reference

## Deliverables

- [ ] `bug-fix.wf.yml` workflow definition
- [ ] Step instructions for each phase
- [ ] Integration with existing `/ace:analyze-bug` and `/ace:fix-bug` workflows

## Acceptance Criteria

- [ ] Workflow loads via `ace-coworker start --workflow bug-fix --issue 123`
- [ ] Each step provides clear instructions to the agent
- [ ] Step transitions require report submission
- [ ] Workflow integrates with GitHub issues (optional `--issue` flag)
