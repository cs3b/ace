# Deprecate ace-git-diff

## Objective

Add deprecation notice to ace-git-diff, keep as thin wrapper for backward compatibility.

## Scope of Work

### Modify ace-git-diff

- Add deprecation warning on load
- Forward all calls to ace-git
- Update README with migration guide
- Update CHANGELOG

### Deprecation Notice

```ruby
# lib/ace/git_diff.rb
warn "[DEPRECATED] ace-git-diff is deprecated. Use ace-git instead."
require "ace/git"

module Ace
  module GitDiff
    # Forward to ace-git for backward compatibility
    Atoms = Ace::Git::Atoms
    Molecules = Ace::Git::Molecules
    Organisms = Ace::Git::Organisms
    Models = Ace::Git::Models
  end
end
```

- [ ] Deprecation warning shown on require
