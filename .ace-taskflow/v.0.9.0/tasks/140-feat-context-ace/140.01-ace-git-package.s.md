# Create ace-git package

## Objective

Create unified ace-git package by merging ace-git-diff functionality with new context/PR/branch features.

## Behavioral Specification

### CLI Interface

```bash
ace-git context              # Repository context (branch, PR, task pattern)
ace-git diff [range]         # Git diff (migrate from ace-git-diff)
ace-git pr [number]          # PR information
ace-git branch               # Branch info only
```

### Output Format (context subcommand)

```markdown
# Repository Context

**Branch:** 140-add-git-context
**Remote:** origin/140-add-git-context (up to date)
**Task Pattern:** 140

## Pull Request

| Field | Value |
|-------|-------|
| Number | #75 |
| Title | Add Git Context |
| Status | open |
| Target | main |
| Author | username |
| URL | https://github.com/owner/repo/pull/75 |
```
