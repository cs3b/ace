# Create ace-context-repo gem

## Objective

Create a new gem that provides pure Git repository context as a composable CLI command.

## Behavioral Specification

### Interface Contract

```bash
# Basic usage
ace-context-repo              # Output as markdown (default)
ace-context-repo --format yaml
ace-context-repo --format json
```

### Output Format (Markdown)

```markdown
# Repository Context

**Branch:** 140-add-git-context
**Remote:** origin/140-add-git-context (up to date | ahead 2 | behind 1)
**Task Pattern:** 140 (extracted from branch)

## Pull Request

| Field | Value |
|-------|-------|
| Number | #75 |
| Title | Add Git Context to ace-context |
| Status | open |
| Target | main |
| Author | username |
| URL | https://github.com/owner/repo/pull/75 |
```

### Error Handling

- Not in git repo: Exit 1, warn message
- Detached HEAD: Show "HEAD (detached)"
- gh not installed: Skip PR section, note in output
- gh not authenticated: Skip PR section, note in output
- No PR for branch: Note "No open pull request"
