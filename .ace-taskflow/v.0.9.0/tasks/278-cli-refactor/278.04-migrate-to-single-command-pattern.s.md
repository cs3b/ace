# ace-git-commit: migrate to single-command pattern

## Scope

Migrate ace-git-commit from registry+DefaultRouting to single-command pattern.

**Binary:** `ace-git-commit`
**CLI module:** `ace-git-commit/lib/ace/git_commit/cli.rb`
**Pattern:** single-command
**Current commands:** commit (only real command)
**Current routing:** DefaultRouting (clean delete, no custom logic)
