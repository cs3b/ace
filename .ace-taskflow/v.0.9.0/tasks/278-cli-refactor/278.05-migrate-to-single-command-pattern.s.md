# ace-test: migrate to single-command pattern

## Scope

Migrate ace-test from registry+DefaultRouting to single-command pattern.

**Binary:** `ace-test`
**CLI module:** `ace-test-runner/lib/ace/test_runner/cli.rb`

**Current commands:** test (only real command)
**Current routing:** DefaultRouting (clean delete, no custom logic)
