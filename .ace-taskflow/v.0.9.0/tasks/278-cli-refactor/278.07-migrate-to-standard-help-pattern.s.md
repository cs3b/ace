# ace-git: migrate to standard help pattern

## Scope

Migrate ace-git from custom start() to standard multi-command help pattern with HelpCommand.

**Binary:** `ace-git`
**CLI module:** `ace-git/lib/ace/git/cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** diff, status, branch, pr
**Current routing:** Custom `start()` with magic git range routing

## Custom Logic to Preserve

The current `start()` has git range magic: args like `HEAD~5..HEAD`, `HEAD~2`, `HEAD`, `@{1}` are auto-prepended with `diff`. This must be preserved — move to exe/ace-git wrapper or a before_callback.

Also preserves: `rescue SystemExit => e` to capture exit status codes.

## Code Changes

- [ ] Remove custom `start()` method
- [ ] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND` constants
- [ ] Add `HELP_EXAMPLES` constant
- [ ] Register HelpCommand for `--help` and `-h`
- [ ] Move git range routing logic to exe/ace-git (pre-process args before `Dry::CLI.new(CLI).call`)
- [ ] Preserve SystemExit rescue in exe/ wrapper
- [ ] No args → show help
