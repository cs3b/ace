# ace-git-secrets: migrate to standard help pattern

## Scope

Migrate ace-git-secrets from custom start() to standard multi-command help pattern.

**Binary:** `ace-git-secrets`
**CLI module:** `ace-git-secrets/lib/ace/git/secrets/cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** scan, rewrite-history, revoke, check-release
**Current routing:** Custom `start()` — preloads config before dispatch

## Custom Logic to Preserve

`start()` calls `Ace::Git::Secrets.config` before routing — for thread safety / TOCTOU on mutex-synchronized config. Move to exe/ wrapper or before_callback.

## Code Changes

- [ ] Remove custom `start()` method
- [ ] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND` constants
- [ ] Add `HELP_EXAMPLES` constant
- [ ] Register HelpCommand for `--help` and `-h`
- [ ] Move config preloading to exe/ace-git-secrets (before `Dry::CLI.new(CLI).call`)
- [ ] No args → show help
