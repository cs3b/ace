# Investigate idea-manager Templates

## Objective

Trace how `ace-taskflow idea create --llm` works and determine if the idea-manager templates in dev-handbook are still needed or should be integrated into the existing capture-idea.wf.md workflow.

## Background

From task 180 discussion: The idea-manager templates in dev-handbook may be obsolete or may contain valuable content that should be integrated with the existing ace-taskflow idea creation workflow.

## Questions to Answer

1. How does `ace-taskflow idea create --llm` currently work?
2. What templates does it use?
3. What's in `dev-handbook/templates/idea-manager/`?
4. Is there overlap with `ace-taskflow/handbook/workflow-instructions/capture-idea.wf.md`?
5. Is the `--llm` flag still used/needed?

## Scope of Work

### Phase 1: Trace Current Implementation

1. Find `ace-taskflow idea create` command implementation
2. Trace `--llm` flag handling
3. Identify template usage

### Phase 2: Analyze dev-handbook/templates/idea-manager/

1. List all files in idea-manager/
2. Review each template's purpose
3. Compare with current ace-taskflow templates

### Phase 3: Recommendations

Based on findings:
- If valuable: Create task to integrate with capture-idea.wf.md
- If obsolete: Mark for skip/archive
- If `--llm` unused: Consider deprecation

## Success Criteria

- [ ] Implementation traced and documented
- [ ] idea-manager templates reviewed
- [ ] Clear recommendation made (integrate/skip)
- [ ] Follow-up task created if needed

## Deliverables

Investigation report with:
- Current implementation overview
- Template comparison
- Recommendation with rationale
