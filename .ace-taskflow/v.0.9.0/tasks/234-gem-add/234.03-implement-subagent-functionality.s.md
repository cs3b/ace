# Subagent support

## Objective

Enable workflow instructions to spawn subagents via Task tool for isolated execution.

## Scope

Enhance workflow instructions to support subagent spawning:
- Instructions can include Task tool invocation syntax
- Agent follows instructions which may include spawning subagents
- Context isolation between parent and subagent

## Deliverables

- [ ] Document subagent instruction pattern
- [ ] Example workflow using subagent spawning
- [ ] Guide for when to use subagents vs direct execution

**Workflow example:**
```yaml
steps:
  - name: implement
    instructions: |
      Use Task tool with subagent_type=general-purpose:
      "Work on task $task following wfi://work-on-task"

  - name: release
    instructions: ace-bundle wfi://draft-release
```

## Acceptance Criteria

- [ ] Instructions can include Task tool spawning
- [ ] Subagent reports captured correctly
- [ ] Context isolation demonstrated
- [ ] Documentation explains when to use subagents
