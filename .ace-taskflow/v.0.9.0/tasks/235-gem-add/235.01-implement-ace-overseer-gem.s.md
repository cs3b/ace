# Implement ace-overseer gem

## Objective

Create ace-overseer - a multi-session orchestrator that manages multiple ace-coworker instances across worktrees. Build after ace-coworker is proven stable.

**Core Value:**
- Spawn and manage multiple coworker sessions
- Coordinate work distribution across git worktrees
- Enable parallel task execution with state tracking

**Principle:** Master multi-session coordination after single session is proven.

## Behavioral Specification

### User Experience

**Input:** User invokes `ace-overseer orchestrate --tasks 121,122,123`

**Process:**
1. Overseer creates worktrees for each task (or reuses existing)
2. Spawns ace-coworker session in each worktree
3. Monitors progress across all sessions
4. Coordinates handoffs and dependencies between tasks
5. Aggregates results and status

**Output:** Multiple tasks completed in parallel with coordinated state

### Interface Contract

```bash
# Start multi-task orchestration
ace-overseer orchestrate --tasks 121,122,123
# Output: Worktrees created, coworkers spawned, progress shown

# Check status of all active sessions
ace-overseer status
# Output: Per-task progress, overall completion percentage

# Stop all active sessions
ace-overseer stop --all
# Output: Sessions gracefully terminated
```

### Expected Behavior

- Automatically creates/reuses git worktrees for task isolation
- Spawns ace-coworker instances with proper context per worktree
- Monitors all sessions and reports aggregate progress
- Handles session failures gracefully with retry options
- Coordinates dependencies between parallel tasks

### Success Criteria

- [ ] `ace-overseer orchestrate --tasks X,Y,Z` creates worktrees and spawns coworkers
- [ ] `ace-overseer status` shows progress across all active sessions
- [ ] Parallel execution works with proper isolation
- [ ] Dependencies between tasks are respected
- [ ] Failed sessions can be retried individually

## Scope of Work

### Deliverables

**Create:**
- `ace-overseer/` gem directory with standard structure
- `lib/ace/overseer/orchestrator.rb` - multi-session coordination
- `lib/ace/overseer/worktree_manager.rb` - git worktree integration
- `exe/ace-overseer` - CLI with orchestrate, status, stop commands

### Capabilities

- Spawn multiple coworkers
- Manage worktrees (via ace-git-worktree integration)
- Distribute work across sessions
- Coordinate parallel execution
- Aggregate status and results

## Out of Scope

- Complex dependency graphs (simple linear dependencies only)
- Cross-repository orchestration
- Cloud/distributed execution
