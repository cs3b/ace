# Add typography validation to ace-lint

## Scope

Add typography validation to ace-lint for enforcing markdown style guidelines from markdown-style.g.md.

### In Scope
- Detect em-dashes (U+2014) and warn to use spaced hyphen
- Detect smart/curly quotes and warn to use straight quotes
- Configurable severity (warn/error/off)
- Tests for typography validation

### Out of Scope
- Autofix functionality (future enhancement)
- Other markdown style rules beyond typography
- Changes to ace-docs

## Behavioral Specification

**User Experience:**
- Input: `ace-lint docs/*.md` command
- Process: Check typography rules per markdown-style.g.md
- Output: Warnings for em-dashes, smart quotes

**Expected Behavior:**
- ace-lint detects em-dashes (U+2014) and warns to use spaced hyphen
- ace-lint detects smart/curly quotes and warns to use straight quotes
- Configurable via `.ace/lint/config.yml` (warn/error/off)

**Interface Contract:**
```bash
# Typography check
ace-lint docs/vision.md
# Output: "Line 42: Em-dash detected. Use ' - ' (spaced hyphen) instead"

# Config (optional)
# .ace/lint/config.yml
typography:
  em_dash: warn      # error | warn | off
  smart_quotes: warn
```
