# Streamline Task Lifecycle: Eliminate Upfront Planning Phase

## Overview

Restructure the task lifecycle from 3 phases (Draft → Plan → Work) to a streamlined flow (Draft → Review & Ready → Work with JIT Planning). The plan-task phase becomes an ephemeral ace-assign phase instead of a persistent lifecycle step. The review-task workflow becomes the gate that promotes drafts to pending.

**Key principles:**
- Plans are ephemeral — JIT planning output goes to ace-assign phase reports, not task specs
- review-task becomes the readiness gate (draft → pending)
- plan-task becomes an ace-assign phase executed just before implementation
- Task specs keep behavioral focus only (WHAT, not HOW)

- **02**: ace-assign Integration (phase catalog, recipes, composition rules)
