## Feature Summary

### 240.01 Human Approval Gates

Pause workflow at critical points for human review:
- `gate: true` in step frontmatter
- `ace-coworker resume --approve` or `--reject`
- Simple rejection handling (mark session, don't rewind)

### 240.02 Multi-Session Support

Multiple concurrent sessions with listing and filtering:
- `ace-coworker list` command
- Session status: active, completed, failed
- `--task` filter to find sessions for a task
- Auto-resume when starting same task again

### 240.03 Frontmatter Schema

Declarative session configuration in task/workflow files:
- `coworker:` frontmatter block in .wf.md files
- Define steps, presets, variables declaratively
- Session hierarchy (parent/child sessions)
- Simplify preset management
