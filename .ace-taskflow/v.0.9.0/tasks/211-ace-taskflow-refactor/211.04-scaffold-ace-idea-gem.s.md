# Scaffold ace-idea gem

## Objective

Create the basic structure for the new `ace-idea` gem following ATOM architecture. This task sets up the gem scaffold before moving code from ace-taskflow.

## Scope

Create gem structure with:
- Standard ATOM directories (atoms, molecules, organisms, models, commands)
- CLI infrastructure with dry-cli
- Configuration with .ace-defaults
- Handbook directory for workflows/agents
- Test infrastructure

## Technical Notes

**Configuration defaults (.ace-defaults/idea/config.yml):**
```yaml
idea:
  root: ".ace-ideas"
  default_release: "inbox"
  directories:
    archive: "_archive"
    maybe: "_maybe"
    anyday: "_anyday"
  defaults:
    git_commit: false
    llm_enhance: false
```

**CLI structure:**
```ruby
module Ace::Idea::CLI
  extend Dry::CLI::Registry

  REGISTERED_COMMANDS = %w[create done park unpark list].freeze
  # ...
end
```
