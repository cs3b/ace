# Create ace-support-git gem

## Objective

Extract git commit operations from ace-taskflow into a new `ace-support-git` gem. This provides reusable git utilities for ace-idea and other gems.

## Scope

Extract and consolidate git commit functionality:
- `ace-taskflow/lib/ace/taskflow/molecules/git_committer.rb` → move to ace-support-git
- `ace-taskflow/lib/ace/taskflow/molecules/git_executor.rb` → deprecate (uses raw Open3)

## Deliverables

### Create
- `ace-support-git/lib/ace/support/git.rb`
- `ace-support-git/lib/ace/support/git/molecules/commit_operation.rb`
- `ace-support-git/lib/ace/support/git/version.rb`
- `ace-support-git/.ace-defaults/git/config.yml`
- `ace-support-git/ace-support-git.gemspec`
- `ace-support-git/test/molecules/commit_operation_test.rb`

### Deprecate
- `ace-taskflow/lib/ace/taskflow/molecules/git_executor.rb` (remove)
- `ace-taskflow/lib/ace/taskflow/molecules/git_committer.rb` (update to use ace-support-git)
