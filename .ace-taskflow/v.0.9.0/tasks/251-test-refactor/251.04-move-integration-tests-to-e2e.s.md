# Move ace-git-secrets Integration Tests to E2E

## Objective

Reduce ace-git-secrets test execution time from 10.16s to ~3-4s by moving git integration tests (which use real git operations via system() calls) to E2E test suite and adding mocks to unit tests.

## Scope of Work

- **Test Optimization**: Move git workflow integration tests to E2E suite
- **Mock Enhancement**: Add git operation mocks to unit tests
- **Performance Target**: 65% reduction in test execution time (~7s saved)

### Deliverables

#### Create
- `ace-git-secrets/test/e2e/MT-SECRETS-001-full-scan-workflow.mt.md`
- `ace-git-secrets/test/e2e/MT-SECRETS-002-rewrite-workflow.mt.md`
- `ace-git-secrets/test/e2e/MT-SECRETS-003-config-cascade.mt.md`

#### Delete
- `ace-git-secrets/test/integration/full_workflow_test.rb` (14 tests)
- `ace-git-secrets/test/integration/config_cascade_test.rb` (5 tests)

#### Modify
- `ace-git-secrets/test/molecules/history_scanner_test.rb` (add git mocks)
- `ace-git-secrets/CHANGELOG.md`
