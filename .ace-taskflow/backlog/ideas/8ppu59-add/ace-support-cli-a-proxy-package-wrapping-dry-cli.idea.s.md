# Idea

ace-support-cli: A proxy package wrapping dry-cli with accumulated patches and extras (ArgvCoalescer, future fixes). Packages would depend on ace-support-cli instead of dry-cli directly, giving us a single place to add workarounds and enhancements. Motivated by the growing number of dry-cli quirks we patch locally (array flag coalescing, help formatting, etc.).
