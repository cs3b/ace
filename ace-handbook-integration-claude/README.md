# ace-handbook-integration-claude

Claude-specific provider integration for ACE handbook skills.

This package is the replacement target for the legacy `ace-integration-claude` package. It owns
Claude-native workflows and provider manifests, while canonical skill definitions remain in the
owning package `handbook/skills` directories and shared sync/runtime behavior lives in `ace-handbook`.
