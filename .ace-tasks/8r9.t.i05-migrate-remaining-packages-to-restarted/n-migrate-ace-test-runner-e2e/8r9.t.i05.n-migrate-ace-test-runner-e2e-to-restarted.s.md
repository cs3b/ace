# Migrate ace-test-runner-e2e to restarted test/e2e and scenario model

## Objective

Migrate ace-test-runner-e2e to the restarted testing structure by moving deterministic sandboxed coverage into ordinary Minitest under test/e2e and applying the pilot scenario decision for this package.

1. Migrate the following deterministic source coverage into test/e2e:
- TS-RUNNER-001-cli-smoke: TC-001-help-command-surface, TC-002-invalid-package-dry-run, TC-003-dry-run-discovers-repo-scenarios, TC-004-suite-help-command-surface

- [ ] Every listed deterministic TC is re-expressed in test/e2e as explicit test methods.
