# ace-overseer README and Getting Started

## Objective

Rewrite ace-overseer's README from 76-line technical overview into a landing page. The current README is closer to the right length but leads with SDK composition details ("composes existing ACE tools through their public Ruby SDKs") instead of user value. It lists internal dependencies (ace-git-worktree, ace-tmux, ace-assign, ace-taskflow, ace-git) before explaining why someone would use it. Create a tutorial-style getting-started.md.

### Expected Behavior

New README should convey: "One command to provision a worktree, open a tmux window, and start an assignment."

Key messaging:

Current README problems to fix:
