# Validate tracer slice to ContextPack

## Behavioral Specification

### User Experience
- **Input**: One representative Markdown document passed to `ace-compressor compress <file> --mode exact`.
- **Process**: The compressor runs the smallest end-to-end slice from source text through parsing, normalization, and emission.
- **Output**: One valid `ContextPack/1` artifact that proves the product shape works before broader feature coverage.

### Expected Behavior
This tracer slice must validate the end-state format and user flow with one realistic document. It is not allowed to hand-wave the final output shape; it must establish the canonical record order, provenance presence, and unresolved-item signaling.

### Interface Contract
```bash
ace-compressor compress docs/vision.md --mode exact
```

**Error Handling:**
- Empty input file: exit non-zero with a clear message.
- Parseable but unsupported embedded image chart: emit unresolved chart marker.

**Edge Cases:**
- Duplicate headings still map to stable section identifiers within the file.
- Empty frontmatter does not block output creation.

### Success Criteria
- [ ] One real Markdown file reaches valid `ContextPack/1` output end to end.
- [ ] Output record ordering is deterministic.
- [ ] Provenance is present for extracted rules/facts.
- [ ] Unresolved items are explicit.

### Validation Questions
- [x] The tracer slice uses a single real document, not fabricated data only.
- [x] The output format proven here becomes the baseline for later exact-mode work.
- [x] Unresolved non-text artifacts remain visible in the tracer output.
- [x] This slice optimizes for end-state validation over broad coverage.

### Vertical Slice Decomposition (Task/Subtask Model)
- **Slice Type**: Subtask
- **Slice Outcome**: A single-file exact compression path that validates final output shape.
- **Advisory Size**: medium
- **Context Dependencies**: archived idea, experiment lessons

### Verification Plan

#### Unit / Component Validation

#### Integration / E2E Validation (if cross-boundary behavior exists)
