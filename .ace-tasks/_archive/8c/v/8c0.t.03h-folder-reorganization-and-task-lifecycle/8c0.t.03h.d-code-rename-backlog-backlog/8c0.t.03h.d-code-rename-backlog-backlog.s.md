### Backward Compatibility

```ruby
def backlog_path
  new_path = File.join(taskflow_root, backlog_dir)
  old_path = File.join(taskflow_root, 'backlog')

  if Dir.exist?(new_path)
    new_path
  elsif Dir.exist?(old_path)
    warn "Deprecated: Using 'backlog/' folder. Rename to '_backlog/' for consistency."
    old_path
  else
    new_path  # Return new path even if it doesn't exist yet
  end
end
```
