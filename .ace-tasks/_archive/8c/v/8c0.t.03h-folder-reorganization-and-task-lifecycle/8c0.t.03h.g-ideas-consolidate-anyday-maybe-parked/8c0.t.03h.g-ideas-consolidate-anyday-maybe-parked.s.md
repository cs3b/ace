# Ideas: Consolidate anyday+maybe → _parked

## Description

Consolidate the `anyday/` and `maybe/` folders for ideas into a single `_parked/` folder. These represent ideas that are "good but not now" - they've been captured but aren't being actively pursued.

## Acceptance Criteria

- [ ] Add `parked_dir` to configuration (default: `_parked`)
- [ ] Add `move_to_parked` method to `IdeaDirectoryMover`
- [ ] Add `idea park <ref>` and `idea unpark <ref>` commands
- [ ] Migrate existing `anyday/` and `maybe/` contents to `_parked/`
- [ ] Add backward compatibility: detect old `anyday/` and `maybe/` folders with warning
- [ ] Update documentation
- [ ] All tests pass

## Backward Compatibility

### Old Folder Detection

When old `anyday/` or `maybe/` folders are detected:

**Warning format**:
```
[DEPRECATED] Found legacy 'anyday/' folder. Run 'ace-taskflow migrate --scope ideas' to consolidate into '_parked/'.
[DEPRECATED] Found legacy 'maybe/' folder. Run 'ace-taskflow migrate --scope ideas' to consolidate into '_parked/'.
```

**Behavior**:
- Warn once per CLI invocation (not per operation)
- Continue to read from old folders if `_parked/` doesn't exist
- New ideas always go to `_parked/` (never create old folders)

### Deprecation Timeline
- **v.0.9.0**: Old folder names deprecated with warnings, migration available
- **v.1.0.0**: Old folder support removed (breaking change)
