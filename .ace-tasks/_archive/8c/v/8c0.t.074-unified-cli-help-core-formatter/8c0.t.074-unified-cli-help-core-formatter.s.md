# Unified CLI Help: Core Formatter

## Overview

Centralize all `--help` / `-h` formatting logic in `ace-support-core` by monkey-patching `Dry::CLI::Banner` and `Dry::CLI::Usage`. This is the foundation phase: once merged, all 20 dry-cli tools automatically get the new two-tier format with zero per-gem changes.

Two-tier behaviour:
- `-h` → compact one-screen summary (name, usage, options only, 2–3 examples, footer)
- `--help` → full reference with ALL-CAPS sections: NAME, USAGE, DESCRIPTION, ARGUMENTS, OPTIONS, EXAMPLES

Top-level registry (`ace-tool --help`) shows first-line summary only (not full multiline desc), with optional command grouping via a `COMMAND_GROUPS` constant on the registry.
