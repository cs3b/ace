# Base36 Compact ID Format Implementation

## Overview

Orchestrator for replacing 14-character timestamps with 6-character Base36 compact IDs across ALL ACE packages. This involves creating a new `ace-support-timestamp` package with configurable encoding, then migrating all affected packages to use the new format. Additionally improves folder grouping for better organization.

## Subtasks

- **01**: Create ace-support-timestamp Package with Base36 Compact IDs (16h)
- **02**: Migrate ace-taskflow to Base36 Compact IDs (6h)
- **03**: Migrate ace-prompt to Base36 Compact IDs (4h)
- **04**: Migrate ace-test-runner to Base36 Compact IDs (4h)
- **05**: Migrate ace-review to Base36 Compact IDs (3h)
- **06**: Migrate ace-docs to Base36 Compact IDs (3h)
- **07**: Migrate ace-git-secrets to Base36 Compact IDs and Group Files (4h)

## Dependencies

- All migration subtasks (02-07) depend on Subtask 01 (package creation)
- All migration subtasks are independent of each other
