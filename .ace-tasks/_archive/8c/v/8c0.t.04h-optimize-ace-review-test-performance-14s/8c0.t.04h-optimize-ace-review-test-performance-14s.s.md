### Key Slow Tests Identified

1. **MultiModelCliTest (21 tests × 0.12s = 2.5s total)**
   - Location: `test/integration/multi_model_cli_test.rb`
   - Issue: Creates real git repo with `git init` but tests don't use git
   - Fix: Remove git setup entirely (CLI parsing tests)

2. **Synthesis Tests (4 tests × 0.8s = 3.2s total)**
   - Location: `test/integration/synthesis_test.rb`, `test/molecules/report_synthesizer_test.rb`
   - Issue: `mock_llm_synthesis` uses class_eval/alias_method (slow)
   - Fix: Use `define_singleton_method` pattern consistently

3. **resolve_prompt_path Test (1 test × 0.47s)**
   - Location: `test/molecules/report_synthesizer_test.rb`
   - Issue: May be doing expensive file resolution
   - Fix: Stub the prompt resolver or cache path resolution
