## Overview

Create ace-coworker - a single session workflow executor that tracks delegations and reports between agent and CLI. Agent is the driver; CLI manages state.

**Core Value:**
- Track what goes to agent (delegation)
- Track what comes back (report)
- Persist state for resume after interruption

**Principle:** Master single session before multi-session.
