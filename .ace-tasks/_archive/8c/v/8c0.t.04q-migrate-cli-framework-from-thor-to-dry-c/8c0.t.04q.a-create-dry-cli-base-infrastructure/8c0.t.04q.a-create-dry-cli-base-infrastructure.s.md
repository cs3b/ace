## Implementation Notes

### Exit Code Handling Pattern

dry-cli commands should return exit codes via the return value of `call`:

```ruby
def call(**options)
  # ... work ...
  0  # Success
rescue StandardError => e
  warn "Error: #{e.message}"
  1  # Failure
end
```

The exe entry point handles the exit:

```ruby
#!/usr/bin/env ruby
require_relative "../lib/ace/gem"
result = Dry::CLI.new(Ace::Gem::CLI).call
exit(result.is_a?(Integer) ? result : 0)
```

### ConfigSummary Integration

```ruby
module Ace::Core::CLI::DryCli
  module ConfigSummaryMixin
    def display_config_summary(command, options)
      return if options[:quiet]

      Ace::Core::Atoms::ConfigSummary.display(
        command: command,
        config: self.class.gem_config,
        defaults: self.class.gem_defaults,
        options: options,
        quiet: false
      )
    end
  end
end

# Usage in Command:
class Create < Dry::CLI::Command
  include Ace::Core::CLI::DryCli::ConfigSummaryMixin

  def call(**options)
    display_config_summary("create", options)
    # ...
  end
end
```
