# Migrate ace-git to dry-cli

## Objective

Migrate ace-git CLI from Thor to dry-cli.

## CLI Analysis

**Complexity**: Moderate
**Commands**: `status` (default), `branch`, `diff`
**Options**: json, no-pr, commits, since

## Scope of Work

#### Modify

- `ace-git/lib/ace/git/cli.rb`
- `ace-git/ace-git.gemspec`
- `ace-git/CHANGELOG.md`

## Implementation Plan

- [ ] Use infrastructure from 179.01:
  - `require "ace/core/cli/dry_cli/base"` for base patterns
  - `Ace::Core::CLI::DryCli::ConfigSummaryMixin` for config display
  - `Ace::Core::CLI::DryCli::VersionCommand` for version handling
- [ ] Convert cli.rb to dry-cli Registry
- [ ] Add dry-cli dependency
- [ ] Run tests: `ace-test ace-git`
- [ ] Verify: `ace-git status`, `ace-git branch`, `ace-git diff`
- [ ] Update CHANGELOG

## Acceptance Criteria

- [ ] All tests pass
- [ ] CLI behavior identical
