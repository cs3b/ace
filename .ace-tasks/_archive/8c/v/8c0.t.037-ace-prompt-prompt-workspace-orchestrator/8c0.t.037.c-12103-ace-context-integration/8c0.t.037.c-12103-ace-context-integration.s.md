# 121.03 - ace-context Integration

## Scope

Optional context loading via frontmatter delegation to ace-context.

## Interface

```bash
ace-prompt --context           # Enable context loading
ace-prompt -c                  # Short flag
ace-prompt --no-context        # Disable context loading
```

## Configuration

```yaml
# .ace/prompt/config.yml
prompt:
  params:
    context: false  # --context / --no-context
```

## Frontmatter Format

```yaml
---
context:
  files:
    - path/to/file.rb
    - src/main.ts
  commands:
    - git diff HEAD~1
    - git status --short
  presets:
    - project
---
[Prompt content here]
```

## Behavior

1. Parse frontmatter for `context` key
2. If `context` present and flag enabled:
   - Delegate to `ace-context --embed-source --stdin`
   - ace-context returns complete output (context + source)
3. If no `context` or flag disabled:
   - Output prompt content as-is
4. Archive happens before context processing

### Edge Cases for Testing

#### FrontmatterExtractor
- No frontmatter (just content)
- Empty frontmatter (`---\n---`)
- Invalid YAML in frontmatter (graceful error)
- Unclosed frontmatter (only opening `---`)
- Frontmatter with complex nested YAML
- Content with `---` in body (not mistaken for frontmatter)
- Empty file
- File with only frontmatter, no body

#### ConfigLoader
- Missing config file (use defaults)
- Partial config (missing keys, merge with defaults)
- Invalid YAML syntax (helpful error)
- Type mismatch (string where hash expected)
- nil values in config (explicit nil wins)
- Deep nested hash merging
- Config immutability (returns new instance each call)
- `params:` section missing (use defaults)

#### ContextLoader
- Missing ace-context gem (graceful fallback, return original)
- ace-context returns error (propagate or fallback)
- Empty context section in frontmatter
- `--no-context` overrides config `context: true`
- `--context` overrides config `context: false`
- Context with invalid file paths (ace-context handles)
- Context with failing commands (ace-context handles)
