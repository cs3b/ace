# ace-prompt: Prompt Workspace (Orchestrator)

## Overview

ace-prompt provides a **Prompt Workspace** - a single active prompt file with automatic history (archive). Write in editor, run `ace-prompt`, get archived output with optional context and enhancement.

**Concept**: One workspace per project (or per task) - not a queue.

**Supersedes**: Tasks 118 and 120 (previous implementation)

## Key Design Decisions

1. **Archive + stdout by default**: Always archive on read, output to stdout
2. **No separate commands**: Only flags (`--enhance`, `--ace-context`)
3. **Full delegation to ace-context**: No merging logic in ace-prompt
4. **Unified archive**: Single archive with `_e001`, `_e002` suffixes for enhancements
