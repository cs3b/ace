# Refactor ace-git to eliminate wrapper pattern

## Objective

Eliminate the wrapper pattern in ace-git by merging dry-cli wrapper classes with their corresponding `*Command` business logic classes. This simplifies the codebase by consolidating 8 files into 4, reducing indirection and improving maintainability while preserving all existing functionality.

## Scope of Work

- **User Experience Scope**: ace-git CLI commands (diff, status, branch, pr)
- **System Behavior Scope**: Command execution, git operations, output formatting
- **Constraint**: Zero functional changes - commands work exactly as before

```

  diff_command.rb   # business logic
  status.rb         # dry-cli wrapper - delegates to StatusCommand
```

## Technical Approach

### Architecture Pattern

The refactoring follows a "consolidation" pattern where thin wrapper classes are merged with their implementation classes. The dry-cli `Dry::CLI::Command` base class provides the interface definition (desc, options, arguments) while the `call` method contains the business logic.

### Merge Strategy

For each command pair (wrapper + implementation):

1. **Keep dry-cli structure** from wrapper file (class inheritance, include, desc, options, arguments, example)
2. **Move business logic** from `*Command` class into the `call` method
3. **Convert instance to class method calls** where applicable (e.g., `execute` becomes inline in `call`)
4. **Preserve private/protected methods** from `*Command` class
5. **Maintain testability patterns** (protected methods for dependency injection)

### Example Transformation

**Before (diff.rb wrapper):**
```ruby
class Diff < Dry::CLI::Command
  include Ace::Core::CLI::DryCli::Base
  desc "Generate git diff..."
  option :format, ...

  def call(range: nil, **options)
    DiffCommand.new.execute(range, options)
  end
end
```

**After (diff.rb consolidated):**
```ruby
class Diff < Dry::CLI::Command
  include Ace::Core::CLI::DryCli::Base
  desc "Generate git diff..."
  option :format, ...

  def call(range: nil, **options)
    # Business logic moved from DiffCommand.execute
    unless Atoms::CommandExecutor.in_git_repo?
      raise Ace::Git::GitError, "Not a git repository..."
    end
    # ... rest of logic
  end

  private
  # Methods moved from DiffCommand
  def build_options(range, cli_options)
    # ...
  end
end
```
