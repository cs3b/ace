# Move deep_merge_arrays to ace-support-core

## Objective

Extract the `deep_merge_arrays` method from `SubjectExtractor` to a shared utility in `ace-support-core`, making it available to other packages that need deep hash merging with array concatenation.

**Source:** google-pro-preview review (item #8)

## Current Implementation

Location: `ace-review/lib/ace/review/molecules/subject_extractor.rb`

```ruby
def deep_merge_arrays(base, overlay)
  result = base.dup
  overlay.each do |key, value|
    if result[key].is_a?(Hash) && value.is_a?(Hash)
      result[key] = deep_merge_arrays(result[key], value)
    elsif result[key].is_a?(Array) && value.is_a?(Array)
      result[key] = normalize_array(result[key] + value)
    elsif result[key].is_a?(Array)
      result[key] = normalize_array(result[key] + [value])
    elsif result.key?(key) && value.is_a?(Array)
      result[key] = normalize_array([result[key]] + value)
    elsif result.key?(key)
      result[key] = normalize_array([result[key], value])
    else
      result[key] = value.is_a?(Array) ? normalize_array(value) : value
    end
  end
  result
end
```

## Proposed Design

### New Module: `Ace::Core::Atoms::DeepMerger`

```ruby
module Ace
  module Core
    module Atoms
      class DeepMerger
        # Merge hashes with array concatenation and deduplication
        # @param base [Hash] Base hash
        # @param overlay [Hash] Overlay hash
        # @param options [Hash] Options
        # @option options [Boolean] :normalize Remove nil/empty, deduplicate
        # @return [Hash] Merged hash (new instance, does not mutate inputs)
        def self.merge(base, overlay, options = {})
          new(options).merge(base, overlay)
        end
      end
    end
  end
end
```

## Implementation Plan

### Technical Approach

**Key Decision: Extend existing `Ace::Core::Atoms::DeepMerger` rather than create new class**

The existing `DeepMerger` module already provides:
- Recursive hash merging
- Array merge strategies (`:replace`, `:concat`, `:union`)
- Immutability (returns new objects)
- `merge_all` for multiple hashes

**Gap Analysis - Features needed from SubjectExtractor's `deep_merge_arrays`:**

| Feature | Existing DeepMerger | SubjectExtractor | Action |
|---------|---------------------|------------------|--------|
| Hash recursion | Yes | Yes | Keep as-is |
| Array concat | Yes (`:concat`) | Yes | Keep as-is |
| Array union | Yes (`:union`) | Yes (with normalization) | Extend |
| Scalar-to-array coercion | No | Yes | Add |
| Blank filtering | No | Yes (`normalize_array`) | Add |
| Mixed type handling | No (overwrites) | Yes (converts to array) | Add |

**Selected Strategy: Add `:coerce_union` array strategy**

A new `:coerce_union` strategy combines:
1. Scalar-to-array coercion (wrap scalars before merging)
2. Union semantics (deduplicate)
3. Normalization (filter nil/empty values)

This maintains backward compatibility - existing `:replace`, `:concat`, `:union` remain unchanged.

### Step 1: Add `:coerce_union` Strategy to DeepMerger

**File:** `ace-support-core/lib/ace/core/atoms/deep_merger.rb`

```ruby
# Deep merge two hashes
# @option options [Symbol] :array_strategy How to handle arrays
#   :replace - Replace base array with overlay (default)
#   :concat - Concatenate arrays
#   :union - Set union (dedupe by value)
#   :coerce_union - Coerce scalars to arrays, union, filter blanks
def merge(base, other, options = {})
  # ... existing nil handling ...

  array_strategy = options[:array_strategy] || :replace

  result = base.dup

  other.each do |key, other_value|
    base_value = result[key]

    result[key] = if base_value.is_a?(Hash) && other_value.is_a?(Hash)
                    merge(base_value, other_value, options)
                  elsif array_strategy == :coerce_union
                    merge_with_coercion(base_value, other_value)
                  elsif base_value.is_a?(Array) && other_value.is_a?(Array)
                    merge_arrays(base_value, other_value, array_strategy)
                  else
                    other_value
                  end
  end

  result
end

# Merge values with scalar-to-array coercion
# @param base_value [Object] Base value (may be array, scalar, or nil)
# @param other_value [Object] Overlay value
# @return [Object] Merged result
def merge_with_coercion(base_value, other_value)
  base_arr = coerce_to_array(base_value)
  other_arr = coerce_to_array(other_value)

  return other_value if base_arr.nil? && !other_value.is_a?(Array)
  return normalize_array(other_arr) if base_arr.nil?
  return normalize_array(base_arr) if other_arr.nil?

  normalize_array(base_arr | other_arr)
end

# Coerce value to array if not nil
# @param value [Object] Value to coerce
# @return [Array, nil] Array or nil if value was nil
def coerce_to_array(value)
  return nil if value.nil?
  value.is_a?(Array) ? value : [value]
end

# Normalize array: remove nil/empty, deduplicate
# @param arr [Array] Array to normalize
# @return [Array] Normalized array
def normalize_array(arr)
  arr.reject { |v| v.nil? || (v.respond_to?(:empty?) && v.empty?) }.uniq
end
```
