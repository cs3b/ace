# ace-llm: migrate to standard help pattern

## Scope

Migrate ace-llm from DefaultRouting to standard multi-command help pattern.

**Binary:** `ace-llm`
**CLI module:** `ace-llm/lib/ace/llm/cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** query, list-providers (also registered as --list-providers)
**Current routing:** DefaultRouting — clean delete, no custom logic

## Code Changes

- [ ] Remove `extend Ace::Core::CLI::DryCli::DefaultRouting`
- [ ] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND`, `BUILTIN_COMMANDS` constants
- [ ] Remove `start()` method
- [ ] Remove `--list-providers` command alias (keep only `list-providers`)
- [ ] Add `HELP_EXAMPLES` constant
- [ ] Register HelpCommand for `--help` and `-h`
- [ ] No args → show help

## HELP_EXAMPLES (curate)

```ruby
HELP_EXAMPLES = [
  ["Query an LLM", "ace-llm query \"Explain this code\""],
  ["Query with model", "ace-llm query --model claude-sonnet \"Summarize\""],
  ["List providers", "ace-llm list-providers"],
].freeze
```
