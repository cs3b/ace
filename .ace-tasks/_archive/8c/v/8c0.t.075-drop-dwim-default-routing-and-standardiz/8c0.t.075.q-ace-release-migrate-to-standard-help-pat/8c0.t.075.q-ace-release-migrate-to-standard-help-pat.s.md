# ace-release: migrate to standard help pattern

## Scope

Migrate ace-release (ReleaseCLI) from DefaultRouting+custom override to standard multi-command help pattern.

**Binary:** `ace-release`
**CLI module:** `ace-taskflow/lib/ace/taskflow/cli/release_cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** list, show
**Current routing:** DefaultRouting + custom `start()` override for cache clearing

## Custom Logic to Preserve

Cache clearing: `ReleaseResolver.clear_cache!` only. Move to exe/ wrapper.

## Code Changes

- [ ] Remove `extend Ace::Core::CLI::DryCli::DefaultRouting`
- [ ] Remove custom `start()` override
- [ ] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND` constants
- [ ] Add `HELP_EXAMPLES` constant
- [ ] Register HelpCommand for `--help` and `-h`
- [ ] Move cache clearing to exe/ace-release
- [ ] No args → show help
