# ace-idea: migrate to standard help pattern

## Scope

Migrate ace-idea (IdeaCLI) from DefaultRouting+custom override to standard multi-command help pattern.

**Binary:** `ace-idea`
**CLI module:** `ace-taskflow/lib/ace/taskflow/cli/idea_cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** list, show, create, done, park, unpark, reschedule
**Current routing:** DefaultRouting + custom `start()` override for cache clearing

## Custom Logic to Preserve

Cache clearing: `TaskLoader.clear_cache!` and `ReleaseResolver.clear_cache!`. Move to exe/ wrapper.

## Code Changes

- [x] Remove `extend Ace::Core::CLI::DryCli::DefaultRouting`
- [x] Remove custom `start()` override
- [x] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND` constants
- [x] Add `HELP_EXAMPLES` constant
- [x] Register HelpCommand for `--help` and `-h`
- [x] Move cache clearing to exe/ace-idea
- [x] No args → show help
