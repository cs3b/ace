# ace-docs: migrate to standard help pattern

## Scope

Migrate ace-docs from DefaultRouting to standard multi-command help pattern.

**Binary:** `ace-docs`
**CLI module:** `ace-docs/lib/ace/docs/cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** status, discover, update, analyze, validate, analyze-consistency
**Current routing:** DefaultRouting — clean delete, no custom logic

## Code Changes

- [ ] Remove `extend Ace::Core::CLI::DryCli::DefaultRouting`
- [ ] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND`, `BUILTIN_COMMANDS` constants
- [ ] Remove `start()` method
- [ ] Add `HELP_EXAMPLES` constant
- [ ] Register HelpCommand for `--help` and `-h`
- [ ] No args → show help
