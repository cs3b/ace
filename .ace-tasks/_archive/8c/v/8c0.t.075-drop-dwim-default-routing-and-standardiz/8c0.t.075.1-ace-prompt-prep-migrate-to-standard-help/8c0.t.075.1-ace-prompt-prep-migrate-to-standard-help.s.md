# ace-prompt-prep: migrate to standard help pattern

## Scope

Migrate ace-prompt-prep from DefaultRouting to standard multi-command help pattern.

**Binary:** `ace-prompt-prep`
**CLI module:** `ace-prompt-prep/lib/ace/prompt_prep/cli.rb`
**Pattern:** multi-command (registry)
**Current commands:** process, setup
**Current routing:** DefaultRouting — clean delete, no custom logic. Has `CLI::Helpers` module (not related to start()).

## Code Changes

- [ ] Remove `extend Ace::Core::CLI::DryCli::DefaultRouting`
- [ ] Remove `KNOWN_COMMANDS`, `DEFAULT_COMMAND`, `BUILTIN_COMMANDS` constants
- [ ] Remove `start()` method
- [ ] Add `HELP_EXAMPLES` constant
- [ ] Register HelpCommand for `--help` and `-h`

- [ ] Keep `CLI::Helpers` module untouched (used by commands, not routing)
