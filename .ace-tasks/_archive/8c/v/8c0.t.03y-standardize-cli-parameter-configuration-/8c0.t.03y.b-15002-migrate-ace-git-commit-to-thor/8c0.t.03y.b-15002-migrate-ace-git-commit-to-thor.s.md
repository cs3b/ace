# 150.02: Migrate ace-git-commit to Thor

## Objective

Convert ace-git-commit from OptionParser CLI to Thor following ADR-018 pattern.

## Current State

- **Framework**: OptionParser (custom, in exe)
- **Entry Point**: `ace-git-commit/exe/ace-git-commit`
- **Lines of Code**: 128
- **Commands**: Single default command (commit)
- **Interface doc**: `_current/usage/ace-git-commit.md`

## Deliverables

### Modify

## Implementation

### Thor CLI

```ruby
# lib/ace/git_commit/cli.rb
require "thor"

module Ace
  module GitCommit
    class CLI < Thor
      desc "commit [FILES...]", "Generate and execute git commit"
      option :intention, type: :string, aliases: "-i", desc: "Context for LLM message"
      option :message, type: :string, aliases: "-m", desc: "Use message directly"
      option :model, type: :string, desc: "Override LLM model"
      option :only_staged, type: :boolean, aliases: "-s", desc: "Commit only staged"
      option :dry_run, type: :boolean, aliases: "-n", desc: "Show what would commit"
      option :debug, type: :boolean, aliases: "-d", desc: "Debug output"
      option :verbose, type: :boolean, desc: "Verbose output"
      option :quiet, type: :boolean, aliases: "-q", desc: "Suppress config summary"
      option :force, type: :boolean, aliases: "-f", desc: "Force operation"
      def commit(*files)
        require_relative "commands/commit_command"
        Commands::CommitCommand.new(files, options).execute
      end
      default_task :commit

      map "--version" => :version  # -v reserved for verbose per usage/ux.md
      desc "version", "Show version"
      def version
        puts "ace-git-commit #{VERSION}"
      end
    end
  end
end
```

### Command Class

```ruby
# lib/ace/git_commit/commands/commit_command.rb
module Ace
  module GitCommit
    module Commands
      class CommitCommand
        def initialize(files, options = {})
          @files = files
          @options = options
        end

        def execute
          display_config_summary

          commit_options = Models::CommitOptions.new(
            files: @files,
            intention: @options[:intention],
            message: @options[:message],
            # ... map other options
          )

          orchestrator = Organisms::CommitOrchestrator.new
          success = orchestrator.execute(commit_options)

          success ? 0 : 1
        rescue GitError => e
          $stderr.puts "Error: #{e.message}"
          1
        end

        private

        def display_config_summary
          return if @options[:quiet]
          Ace::Core::Atoms::ConfigSummary.display(
            command: "commit",
            config: Ace::GitCommit.config,
            defaults: Ace::GitCommit.default_config,
            options: @options
          )
        end
      end
    end
  end
end
```
