# 150.07: Migrate ace-search to Thor

## Objective

Convert ace-search from OptionParser CLI to Thor following ADR-018 pattern.

## Current State

- **Framework**: OptionParser (custom, in exe)
- **Entry Point**: `ace-search/exe/ace-search`
- **Lines of Code**: 272
- **Commands**: search (default)
- **Interface doc**: `_current/usage/ace-search.md`

## Key Challenges

- Many options (20+)
- DWIM pattern analysis (auto-detect search type)
- Preset merging
- Git scope options (staged, tracked, changed)

## Deliverables

### Create

1. `ace-search/lib/ace/search/cli.rb` - Thor CLI class
2. `ace-search/lib/ace/search/commands/search_command.rb`

### Modify

1. `ace-search/exe/ace-search` - Thor start wrapper
2. `.ace-defaults/search/config.yml` - Add cli_defaults

require "thor"

module Ace
  module Search
    class CLI < Thor
      desc "search PATTERN [PATH]", "Search files and content"
      option :type, type: :string, enum: %w[file content hybrid auto], aliases: "-t"
      option :files, type: :boolean, aliases: "-f", desc: "Files only"
      option :content, type: :boolean, aliases: "-c", desc: "Content only"
      option :case_insensitive, type: :boolean, aliases: "-i"
      option :whole_word, type: :boolean, aliases: "-w"
      option :multiline, type: :boolean, aliases: "-U"
      option :hidden, type: :boolean
      option :after_context, type: :numeric, aliases: "-A"
      option :before_context, type: :numeric, aliases: "-B"
      option :context, type: :numeric, aliases: "-C"
      option :glob, type: :string, aliases: "-g"
      # NOTE: These accept comma-separated values (--include a,b,c)
      option :include, type: :string, desc: "Paths to include (comma-separated)"
      option :exclude, type: :string, aliases: "-e", desc: "Paths to exclude (comma-separated)"
      option :since, type: :string
      option :before, type: :string
      option :staged, type: :boolean
      option :tracked, type: :boolean
      option :changed, type: :boolean
      option :json, type: :boolean
      option :yaml, type: :boolean
      option :files_with_matches, type: :boolean, aliases: "-l"
      option :max_results, type: :numeric
      option :interactive, type: :boolean, desc: "fzf selection"
      option :preset, type: :string, aliases: "-p"
      option :verbose, type: :boolean, aliases: "-v"
      option :quiet, type: :boolean, aliases: "-q"
      def search(pattern, path = nil)
        require_relative "commands/search_command"
        Commands::SearchCommand.new(pattern, path, options).execute
      end
      default_task :search

      map %w[--version] => :version
      desc "version", "Show version"
      def version
        puts "ace-search #{VERSION}"
      end
    end
  end

### Search Command

```ruby
# lib/ace/search/commands/search_command.rb
module Ace
  module Search
    module Commands
      class SearchCommand
        def initialize(pattern, path, options)
          @pattern = pattern
          @path = path
          @options = build_options(options)
        end

        def execute
          display_config_summary

          # Apply preset if specified
          apply_preset if @options[:preset]

          # Resolve search path
          resolver = Atoms::SearchPathResolver.new
          @options[:search_path] = resolver.resolve(@path)

          # Execute search
          searcher = Organisms::UnifiedSearcher.new
          result = searcher.search(@pattern, @options)

          handle_result(result)
        end

        private

        def build_options(thor_options)
          config = Ace::Search.config
          {
            type: thor_options[:type]&.to_sym || config["type"]&.to_sym || :auto,
            format: thor_options[:json] ? :json : (thor_options[:yaml] ? :yaml : :text),
            # ... map all options
          }
        end

        def apply_preset
          preset_manager = Molecules::PresetManager.new
          @options = preset_manager.merge_with_options(@options[:preset], @options)
        end
      end
    end
  end
end
```
