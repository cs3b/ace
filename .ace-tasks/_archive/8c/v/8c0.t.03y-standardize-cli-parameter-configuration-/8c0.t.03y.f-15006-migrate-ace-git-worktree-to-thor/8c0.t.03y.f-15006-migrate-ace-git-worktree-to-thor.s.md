# 150.06: Migrate ace-git-worktree to Thor

## Objective

Convert ace-git-worktree from custom case/when CLI to Thor following ADR-018 pattern.

## Current State

- **Framework**: Custom case/when routing
- **Entry Point**: `ace-git-worktree/lib/ace/git/worktree/cli.rb`
- **Lines of Code**: 196
- **Commands**: create, list, switch, remove, prune, config
- **Interface doc**: `_current/usage/ace-git-worktree.md`

## Key Advantage

Already has Commands:: pattern - straightforward Thor wrapper.

## Deliverables

### Modify

1. `ace-git-worktree/lib/ace/git/worktree/cli.rb` - Convert to Thor
2. `ace-git-worktree/exe/ace-git-worktree` - Thor start wrapper
3. `.ace-defaults/git-worktree/config.yml` - Add cli_defaults

Existing command classes can be reused with minimal changes.

## Implementation

### Thor CLI

```ruby
# lib/ace/git/worktree/cli.rb
require "thor"

module Ace
  module Git
    module Worktree
      class CLI < Thor
        desc "create", "Create new worktree"
        option :task, type: :string, desc: "Task ID for naming"
        option :branch, type: :string, desc: "Branch name"
        option :path, type: :string, desc: "Worktree path"
        option :quiet, type: :boolean, aliases: "-q"
        def create
          display_config_summary
          require_relative "commands/create_command"
          Commands::CreateCommand.new(options).execute
        end

        desc "list", "List all worktrees"
        option :show_tasks, type: :boolean, desc: "Show task info"
        option :json, type: :boolean, desc: "JSON output"
        option :quiet, type: :boolean, aliases: "-q"
        def list
          display_config_summary
          require_relative "commands/list_command"
          Commands::ListCommand.new(options).execute
        end
        map "ls" => :list

        desc "switch REF", "Switch to worktree"
        option :quiet, type: :boolean, aliases: "-q"
        def switch(ref)
          display_config_summary
          require_relative "commands/switch_command"
          Commands::SwitchCommand.new(ref, options).execute
        end
        map "cd" => :switch

        desc "remove", "Remove worktree"
        option :task, type: :string, desc: "Task ID"
        option :force, type: :boolean, aliases: "-f"
        option :quiet, type: :boolean, aliases: "-q"
        def remove
          display_config_summary
          require_relative "commands/remove_command"
          Commands::RemoveCommand.new(options).execute
        end
        map "rm" => :remove

        desc "prune", "Clean up deleted worktrees"
        option :quiet, type: :boolean, aliases: "-q"
        def prune
          display_config_summary
          require_relative "commands/prune_command"
          Commands::PruneCommand.new(options).execute
        end

        desc "config", "Show configuration"
        def config
          require_relative "commands/config_command"
          Commands::ConfigCommand.new(options).execute
        end

        map "--version" => :version  # -v reserved for verbose per usage/ux.md
        desc "version", "Show version"
        def version
          puts "ace-git-worktree #{VERSION}"
        end

        private

        def display_config_summary
          return if options[:quiet]
          Ace::Core::Atoms::ConfigSummary.display(
            command: current_command_chain.join(" "),
            config: Ace::Git::Worktree.config,
            defaults: Ace::Git::Worktree.default_config,
            options: options
          )
        end
      end
    end
  end
end
```

## Command Aliases

Thor `map` handles aliases:
- `ls` → `list`
- `cd` → `switch`
- `rm` → `remove`
