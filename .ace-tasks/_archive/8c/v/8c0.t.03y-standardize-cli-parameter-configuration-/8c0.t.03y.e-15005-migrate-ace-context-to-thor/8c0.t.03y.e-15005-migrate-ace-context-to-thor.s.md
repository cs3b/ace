# 150.05: Migrate ace-context to Thor

## Objective

Convert ace-context from OptionParser CLI to Thor following ADR-018 pattern.

## Current State

- **Framework**: OptionParser (custom, in exe)
- **Entry Point**: `ace-context/exe/ace-context`
- **Lines of Code**: 224
- **Commands**: load (default), list, inspect
- **Interface doc**: `_current/usage/ace-context.md`

## Key Challenges

- Multiple input modes (preset, file, protocol)
- Repeatable options (--preset, --file)
- Auto-format threshold logic
- Output mode handling (stdio, cache, file)

## Deliverables

### Create

1. `ace-context/lib/ace/context/cli.rb` - Thor CLI class
2. `ace-context/lib/ace/context/commands/load_command.rb`
3. `ace-context/lib/ace/context/commands/list_command.rb`
4. `ace-context/lib/ace/context/commands/inspect_command.rb`

### Modify

1. `ace-context/exe/ace-context` - Thor start wrapper
2. `.ace-defaults/context/config.yml` - Add cli_defaults

## Implementation

### Thor CLI

```ruby
# lib/ace/context/cli.rb
require "thor"

module Ace
  module Context
    class CLI < Thor
      desc "load [INPUT]", "Load context from preset, file, or protocol"
      # NOTE: Use repeatable: true (Thor v1.3+) to preserve -p a -p b syntax
      option :preset, type: :string, repeatable: true, aliases: "-p", desc: "Presets to load"
      option :file, type: :string, repeatable: true, aliases: "-f", desc: "Files to load"
      option :output, type: :string, aliases: "-o", desc: "Output mode: stdio, cache, path"
      option :format, type: :string, enum: %w[markdown yaml xml markdown-xml json]
      option :embed_source, type: :boolean, aliases: "-e", desc: "Embed source in output"
      option :max_size, type: :numeric, desc: "Max file size (bytes)"
      option :timeout, type: :numeric, desc: "Timeout (seconds)"
      option :debug, type: :boolean, aliases: "-d"
      option :quiet, type: :boolean, aliases: "-q"
      def load(input = nil)
        require_relative "commands/load_command"
        Commands::LoadCommand.new(input, options).execute
      end
      default_task :load

      desc "list", "List available presets"
      def list
        require_relative "commands/list_command"
        Commands::ListCommand.new(options).execute
      end
      map %w[-l --list --list-presets] => :list

      desc "inspect [INPUT]", "Inspect config without execution"
      option :preset, type: :string, repeatable: true, aliases: "-p"
      option :file, type: :string, repeatable: true, aliases: "-f"
      def inspect_config(input = nil)
        require_relative "commands/inspect_command"
        Commands::InspectCommand.new(input, options).execute
      end
      map "--inspect-config" => :inspect_config

      map "--version" => :version  # -v reserved for verbose per usage/ux.md
      desc "version", "Show version"
      def version
        puts "ace-context #{VERSION}"
      end
    end
  end
end
```

### Load Command

```ruby
# lib/ace/context/commands/load_command.rb
module Ace
  module Context
    module Commands
      class LoadCommand
        def initialize(input, options)
          @input = input
          @options = options
        end

        def execute
          display_config_summary

          # Collect inputs
          inputs = collect_inputs

          # Load context
          context = if inputs.size > 1
                      Ace::Context.load_multiple_inputs(
                        @options[:preset] || [],
                        @options[:file] || [],
                        @options
                      )
                    else
                      Ace::Context.load_auto(inputs.first, @options)
                    end

          # Handle errors
          handle_error(context) if context.metadata[:error]

          # Determine output mode
          output_mode = determine_output_mode(context)

          # Output
          output_content(context, output_mode)

          0
        end

        private

        def collect_inputs
          inputs = []
          inputs.concat(@options[:preset] || [])
          inputs.concat(@options[:file] || [])
          inputs << @input if @input
          inputs << "default" if inputs.empty?
          inputs
        end

        def determine_output_mode(context)
          return @options[:output] if @options[:output]
          return context.metadata[:output] if context.metadata[:output]

          # Auto-format based on line count
          line_count = Atoms::LineCounter.count(context.content)
          line_count >= Ace::Context.auto_format_threshold ? "cache" : "stdio"
        end
      end
    end
  end
end
```
