# 150.10: Migrate ace-test-runner to Thor

## Objective

Convert ace-test-runner from OptionParser CLI to Thor following ADR-018 pattern.

## Current State

- **Framework**: OptionParser (complex with CliArgumentParser molecule)
- **Entry Point**: `ace-test-runner/exe/ace-test`
- **Lines of Code**: 312
- **Commands**: test (default), cleanup, rake integration, fix
- **Interface doc**: `_current/usage/ace-test.md`

## Key Challenges

- **CliArgumentParser molecule**: Complex argument disambiguation
- **Special exit handling**: Must use `exit!` not `exit` (Minitest compatibility)
- **Multiple operation modes**: test, cleanup, rake, fix
- **Positional arg flexibility**: package, target, files in any order

## Recommended Approach

Keep CliArgumentParser for positional arg parsing, wrap in Thor for options.

## Deliverables

### Create

1. `ace-test-runner/lib/ace/test_runner/cli.rb` - Thor CLI class
2. `ace-test-runner/lib/ace/test_runner/commands/run_command.rb`
3. `ace-test-runner/lib/ace/test_runner/commands/cleanup_command.rb`
4. `ace-test-runner/lib/ace/test_runner/commands/rake_command.rb`
5. `ace-test-runner/lib/ace/test_runner/commands/fix_command.rb`

### Modify

1. `ace-test-runner/exe/ace-test` - Thor start wrapper with exit! handling
2. `.ace-defaults/test-runner/config.yml` - Add cli_defaults

## Implementation

### Thor CLI

```ruby
# lib/ace/test_runner/cli.rb
require "thor"

module Ace
  module TestRunner
    class CLI < Thor
      class_option :quiet, type: :boolean, aliases: "-q", desc: "Suppress config summary"
      class_option :verbose, type: :boolean, aliases: "-v", desc: "Detailed output"

      desc "run [PACKAGE] [TARGET] [FILES...]", "Run tests"
      option :format, type: :string, aliases: "-f", enum: %w[progress progress-file json]
      option :report_dir, type: :string, aliases: "-d"
      option :save_reports, type: :boolean, default: true
      option :fail_fast, type: :boolean
      option :filter, type: :string
      option :group, type: :string, aliases: "-g"
      option :color, type: :boolean, default: true
      option :config_path, type: :string, aliases: "-c"
      option :timeout, type: :numeric
      option :parallel, type: :boolean
      option :per_file, type: :boolean
      option :direct, type: :boolean
      option :subprocess, type: :boolean
      option :max_display, type: :numeric
      option :profile, type: :numeric
      def run(*args)
        require_relative "commands/run_command"
        Commands::RunCommand.new(args, options).execute
      end
      default_task :run

      desc "cleanup", "Clean up old test reports"
      option :keep, type: :numeric, default: 10
      option :age, type: :numeric, default: 30
      def cleanup
        require_relative "commands/cleanup_command"
        Commands::CleanupCommand.new(options).execute
      end
      map "--cleanup-reports" => :cleanup

      desc "rake ACTION", "Manage rake integration"
      def rake(action)
        require_relative "commands/rake_command"
        Commands::RakeCommand.new(action, options).execute
      end
      map "--set-default-rake-test" => -> { rake("set") }
      map "--unset-default-rake-test" => -> { rake("unset") }
      map "--check-rake-test" => -> { rake("check") }

      desc "fix", "Auto-fix deprecated test patterns"
      def fix
        require_relative "commands/fix_command"
        Commands::FixCommand.new(options).execute
      end
      map "--fix-deprecations" => :fix

      map %w[--version] => :version
      desc "version", "Show version"
      def version
        puts "ace-test-runner #{VERSION}"
      end
    end
  end
end
```

### Run Command

```ruby
# lib/ace/test_runner/commands/run_command.rb
module Ace
  module TestRunner
    module Commands
      class RunCommand
        def initialize(args, options)
          @args = args
          @options = options.to_h.transform_keys(&:to_sym)
        end

        def execute
          display_config_summary

          # Use CliArgumentParser for positional arg disambiguation
          arg_parser = Molecules::CliArgumentParser.new(@args)
          parsed = arg_parser.parse
          @options.merge!(parsed)

          # Execute tests
          exit_code = Ace::TestRunner.run(@options)

          # Return exit code - exe wrapper handles exit!
          exit_code
        rescue ArgumentError => e
          $stderr.puts "Error: #{e.message}"
          1
        end

        private

        def display_config_summary
          return if @options[:quiet]
          Ace::Core::Atoms::ConfigSummary.display(
            command: "run",
            config: Ace::TestRunner.config,
            defaults: Ace::TestRunner.default_config,
            options: @options
          )
        end
      end
    end
  end
end
```

### Executable with exit! Handling

```ruby
#!/usr/bin/env ruby
# frozen_string_literal: true

# Prevent Minitest from autorunning
ENV["MT_NO_AUTORUN"] = "1"

require "ace/test_runner"

begin
  result = Ace::TestRunner::CLI.start(ARGV)
  exit_code = result.is_a?(Integer) ? result : 0

  $stdout.flush
  $stderr.flush

  # CRITICAL: Use exit! to skip at_exit handlers (Minitest)
  exit! exit_code
rescue Ace::TestRunner::Error => e
  $stderr.puts "Error: #{e.message}"
  exit! 1
rescue Interrupt
  $stderr.puts "\nTest execution interrupted"
  exit! 130
end
```

## CliArgumentParser Integration

CliArgumentParser handles:
- Package detection (`ace-*`, `./path`, `/path`)
- Target detection (`atoms`, `molecules`, etc.)
- File detection (`test/*.rb`, `file.rb:42`)
- Pattern disambiguation

Thor varargs pass to CliArgumentParser for parsing.
