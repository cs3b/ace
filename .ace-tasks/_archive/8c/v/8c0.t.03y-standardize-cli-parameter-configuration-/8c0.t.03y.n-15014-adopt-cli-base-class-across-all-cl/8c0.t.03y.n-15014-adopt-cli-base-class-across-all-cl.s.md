# 150.14: Adopt CLI Base Class Across All CLIs

## Objective

Migrate all 13 CLI classes to inherit from `Ace::Core::CLI::Base` instead of `Thor` directly, removing redundant code.

## Deliverables

### Standard Changes Per CLI

1. Add `require "ace/core/cli/base"` at top
2. Change `class CLI < Thor` to `class CLI < Ace::Core::CLI::Base`
3. Remove local `exit_on_failure?` method (inherited from Base)
4. Remove redundant class_options (quiet, verbose, debug) if duplicating Base
5. Remove redundant `respond_to_missing?` if identical to Base
6. Remove `-v` version mapping if present (use `--version` only)
