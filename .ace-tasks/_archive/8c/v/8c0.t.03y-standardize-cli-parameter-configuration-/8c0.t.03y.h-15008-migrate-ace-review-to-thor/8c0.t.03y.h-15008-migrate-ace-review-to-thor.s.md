# 150.08: Migrate ace-review to Thor

## Objective

Convert ace-review from OptionParser CLI to Thor following ADR-018 pattern.

## Current State

- **Framework**: OptionParser (custom CLI class with subcommand)
- **Entry Point**: `ace-review/lib/ace/review/cli.rb`
- **Lines of Code**: 524
- **Commands**: review (default), synthesize
- **Interface doc**: `_current/usage/ace-review.md`

## Key Challenges

- **synthesize subcommand**: Needs Thor subcommand or separate command
- **Repeatable options**: `--subject` and `--model` can be used multiple times
- **Boolean defaults**: `--save-session` defaults to true
- **Complex success handling**: Multi-model results, comment posting

## Deliverables

### Create

1. `ace-review/lib/ace/review/commands/review_command.rb`
2. `ace-review/lib/ace/review/commands/synthesize_command.rb`
3. `ace-review/lib/ace/review/commands/list_presets_command.rb`
4. `ace-review/lib/ace/review/commands/list_prompts_command.rb`

### Modify

1. `ace-review/lib/ace/review/cli.rb` - Convert to Thor
2. `ace-review/exe/ace-review` - Thor start wrapper
3. `.ace-defaults/review/config.yml` - Add cli_defaults

## Implementation

### Thor CLI

```ruby
# lib/ace/review/cli.rb
require "thor"

module Ace
  module Review
    class CLI < Thor
      class_option :quiet, type: :boolean, aliases: "-q", desc: "Suppress config summary"
      class_option :verbose, type: :boolean, aliases: "-v", desc: "Verbose output"

      desc "review", "Execute review using presets or custom configuration"
      option :preset, type: :string
      option :output_dir, type: :string
      option :output, type: :string
      option :context, type: :string
      # NOTE: Use repeatable: true (Thor v1.3+) to preserve --subject a --subject b syntax
      option :subject, type: :string, repeatable: true, desc: "Subject configs (repeatable)"
      option :prompt_base, type: :string
      option :prompt_format, type: :string
      option :prompt_focus, type: :string
      option :add_focus, type: :string
      option :prompt_guidelines, type: :string
      option :model, type: :string, repeatable: true, desc: "LLM models (repeatable)"
      option :no_synthesize, type: :boolean
      option :synthesis_model, type: :string
      option :dry_run, type: :boolean
      option :auto_execute, type: :boolean
      option :save_session, type: :boolean, default: true
      option :session_dir, type: :string
      option :task, type: :string
      option :no_auto_save, type: :boolean
      option :pr, type: :string
      option :pr_comments, type: :boolean, default: true
      option :post_comment, type: :boolean
      option :gh_timeout, type: :numeric
      def review
        require_relative "commands/review_command"
        Commands::ReviewCommand.new(options).execute
      end
      default_task :review

      desc "synthesize", "Synthesize multiple review reports"
      option :session, type: :string, desc: "Session directory"
      option :reports, type: :array, desc: "Report files"
      option :synthesis_model, type: :string
      option :output, type: :string
      def synthesize
        require_relative "commands/synthesize_command"
        Commands::SynthesizeCommand.new(options).execute
      end

      desc "list_presets", "List available presets"
      def list_presets
        require_relative "commands/list_presets_command"
        Commands::ListPresetsCommand.new(options).execute
      end
      map "--list-presets" => :list_presets

      desc "list_prompts", "List available prompt modules"
      def list_prompts
        require_relative "commands/list_prompts_command"
        Commands::ListPromptsCommand.new(options).execute
      end
      map "--list-prompts" => :list_prompts

      map %w[--version] => :version
      desc "version", "Show version"
      def version
        puts "ace-review #{VERSION}"
      end
    end
  end
end
```

### Review Command

```ruby
# lib/ace/review/commands/review_command.rb
module Ace
  module Review
    module Commands
      class ReviewCommand
        def initialize(options)
          @options = normalize_options(options)
        end

        def execute
          display_config_summary

          # Convert to ReviewOptions
          review_options = Models::ReviewOptions.new(@options)

          manager = Organisms::ReviewManager.new
          result = manager.execute_review(review_options)

          if result[:success]
            handle_success(result)
            0
          else
            handle_error(result)
            1
          end
        end

        private

        def normalize_options(options)
          opts = options.to_h.transform_keys(&:to_sym)

          # Handle subject array → merged subject
          if opts[:subject]&.any?
            opts[:subject].uniq!
            opts[:subject] = opts[:subject].size == 1 ? opts[:subject].first : opts[:subject]
          end

          # Handle model array
          opts[:models] = opts[:model] if opts[:model]

          opts
        end

        def handle_success(result)
          # Multi-model handling
          if result[:summary]
            handle_multi_model_success(result)
          else
            # ... single model success
          end
        end
      end
    end
  end
end
```

## Subject Handling

Thor `repeatable: true` preserves OptionParser accumulation behavior:
```bash
# Current (works)
ace-review --subject diff:HEAD~3 --subject files:docs/**/*.md

# Thor with repeatable: true (works same way)
ace-review --subject diff:HEAD~3 --subject files:docs/**/*.md
```

**Note**: Requires Thor v1.3+. The `repeatable: true` option collects repeated flags into an array.

## Backward Compatibility
