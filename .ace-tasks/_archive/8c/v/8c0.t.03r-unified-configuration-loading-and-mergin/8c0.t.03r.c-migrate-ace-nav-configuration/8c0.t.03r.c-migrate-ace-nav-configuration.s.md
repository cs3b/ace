# Migrate ace-nav Configuration

## Objective

Replace the `DEFAULT_CONFIG` hash in `config_loader.rb` with the standard `Ace::Core.config.get()` cascade pattern.

## Current State (Anti-Pattern)

**File: `ace-nav/lib/ace/nav/molecules/config_loader.rb:14-28`**
```ruby
DEFAULT_CONFIG = {
  "cache" => { "enabled" => false, "directory" => ".cache/ace-nav", "ttl" => 3600 },
  "fuzzy" => { "enabled" => true, "threshold" => 0.6 },
  "output" => { "color" => true, "verbose" => false }
}.freeze
```

**Problem**: Default configuration is hardcoded in Ruby instead of being loaded from `.ace.example/nav/config.yml`.

## Target State (Standard Pattern)

```ruby
def self.config
  @config ||= Ace::Core.config.get('ace', 'nav') || load_gem_defaults
end
```
