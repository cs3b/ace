## Objective

Replace duplicated Git/GitHub code in ace-review with ace-git dependency. This eliminates code duplication and centralizes Git operations in the ace-git package.

## Technical Approach

### Architecture Pattern

- **Pattern**: Dependency replacement (using ace-git atoms/molecules)
- **Rationale**: ace-git provides unified Git operations across all ACE packages
- **Integration**: Replace local implementations with ace-git equivalents
- **Impact**: Removes ~4 local files, updates ~5 dependent files

### Technology Stack

- **New Dependency**: ace-git (~> 0.3) - unified Git/GitHub operations
- **Removed Dependency**: ace-git-diff (transitive through ace-git)
- **Unchanged**: ace-context, ace-support-core, ace-nav, ace-llm, ace-taskflow

### Implementation Strategy

1. **Add Dependency**: Update gemspec to depend on ace-git (~> 0.3)
2. **Update Imports**: Change require statements to use ace-git modules
3. **Update Callers**: Replace class references with ace-git equivalents
4. **Remove Files**: Delete local implementations after migration
5. **Update Tests**: Modify test stubs to reference ace-git classes
