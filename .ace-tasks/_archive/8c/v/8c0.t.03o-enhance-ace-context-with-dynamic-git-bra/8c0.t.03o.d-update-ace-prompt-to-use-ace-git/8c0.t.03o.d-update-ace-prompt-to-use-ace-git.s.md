## Objective

Replace duplicated git_branch_reader with ace-git dependency. This eliminates code duplication by using the consolidated `Ace::Git::Molecules::BranchReader` class from ace-git instead of maintaining a local copy in ace-prompt.

## Technical Approach

### Architecture Pattern

- **Pattern**: Dependency consolidation via gem reference
- **Rationale**: ace-git (task 140.01) consolidates all git-related functionality into a single package
- **Integration**: Replace local molecule with ace-git require
- **Impact**: Minimal - only changes internal require path and class reference

### Technology Stack

- **New Dependency**: ace-git (~> 0.3) - provides `BranchReader` molecule
- **Compatibility**: ace-git requires same Ruby version (>= 3.2.0) and ace-support-core (~> 0.11)
- **No Breaking Changes**: API remains identical (`current_branch` class method)

### Implementation Strategy

1. **Add Dependency**: Update gemspec to require ace-git
2. **Update Require**: Change from relative molecule to gem require
3. **Update Reference**: Change namespace from `Ace::Prompt::Molecules::GitBranchReader` to `Ace::Git::Molecules::BranchReader`
4. **Remove Duplicates**: Delete local git_branch_reader.rb and its test
5. **Verify**: Run test suite to confirm no regressions
