# Update ace-search to use ace-git

## Objective

Replace duplicated git_scope_filter in ace-search with ace-git dependency. This eliminates code duplication and centralizes Git file scope operations.

## Technical Approach

### Architecture Pattern

- **Pattern**: Dependency replacement (using ace-git atoms)
- **Rationale**: ace-git provides unified Git operations across all ACE packages
- **Integration**: Replace local GitScopeFilter with ace-git equivalent
- **Impact**: Removes 1 local file, updates ~2 dependent files

### Implementation Strategy

1. **Add Dependency**: Update gemspec to depend on ace-git (~> 0.3)
2. **Update Imports**: Change require statements to use ace-git modules
3. **Update Callers**: Replace class references with ace-git equivalents
4. **Remove Files**: Delete local implementation after migration
5. **Update Tests**: Modify test stubs to reference ace-git classes
