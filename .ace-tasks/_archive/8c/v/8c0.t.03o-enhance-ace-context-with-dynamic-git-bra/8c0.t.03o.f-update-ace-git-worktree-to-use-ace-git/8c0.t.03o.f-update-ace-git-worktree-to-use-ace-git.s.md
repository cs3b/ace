## Objective

Replace duplicated Git/GitHub functionality in ace-git-worktree with the consolidated ace-git package. This reduces code duplication and centralizes all Git operations in a single, well-tested package. The ace-git package (created in task 140.01) provides GitCommand, TaskPatternExtractor, and PrMetadataFetcher that can replace the local implementations.

## Technical Approach

### Architecture Pattern

- **Pattern**: Dependency delegation to ace-git
- **Rationale**: Centralizing Git operations in ace-git prevents duplication across packages
- **Integration**: Replace internal atoms/molecules with ace-git equivalents
- **Impact**: Minimal API changes - internal refactoring only

### Technology Stack

- **Dependencies**:
  - ace-git (~> 0.3.0) - New dependency providing Git operations
  - ace-support-core (~> 0.11) - Already present
  - ace-taskflow (~> 0.10) - Already present
- **Ruby Version**: >= 3.0.0 (unchanged)

### Implementation Strategy

1. **Add Dependency**: Add ace-git to gemspec, remove direct ace-git-diff dependency
2. **Update Requires**: Replace local requires with ace-git requires
3. **Update Code**: Change class references from local to ace-git namespace
4. **Remove Duplicates**: Delete the files that are now provided by ace-git
5. **Update Tests**: Update test imports and assertions to use ace-git
