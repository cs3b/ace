# Update ace-docs to use ace-git

## Objective

Update ace-docs to use ace-git instead of ace-git-diff. This is a minimal change since ace-docs already properly delegates to ace-git-diff - only namespace updates are needed.

## Technical Approach

### Architecture Pattern

- **Pattern**: Namespace update (ace-git-diff → ace-git)
- **Rationale**: ace-git-diff is being deprecated and merged into ace-git
- **Integration**: Update require and class references
- **Impact**: Minimal - only namespace changes in 1 file

### Current State

ace-docs `ChangeDetector` already correctly delegates to ace-git-diff:
```ruby
require "ace/git_diff"
# ...
result = Ace::GitDiff::Organisms::DiffOrchestrator.generate(diff_options)
```

This is the ideal pattern - just needs namespace update.

### Implementation Strategy

1. **Update Dependency**: Replace ace-git-diff with ace-git in gemspec
2. **Update Requires**: Change `require "ace/git_diff"` to `require "ace/git"`
3. **Update References**: Change `Ace::GitDiff::` to `Ace::Git::`
4. **Verify**: Run tests to ensure functionality unchanged
