## Objective

Replace duplicated Git/GitHub code in ace-context with ace-git dependency. This reduces code duplication, centralizes Git operations, and simplifies ace-context's responsibility to focus on context loading rather than Git operations.

## Technical Approach

### Architecture Pattern

- **Pattern**: Dependency Delegation (replace internal components with external package)
- **Rationale**: ace-git consolidates Git/GitHub operations across the monorepo
- **Integration**: Replace internal atoms/molecules with ace-git equivalents
- **Impact**: Reduced code in ace-context, single source of truth for Git operations

### Technology Stack

- **Dependencies**:
  - ace-git (~> 0.3.0) - replaces internal Git components and ace-git-diff
  - ace-support-core (~> 0.10) - unchanged
  - Remove: ace-git-diff (~> 0.1.0)
- **Ruby Version**: >= 3.2.0 (unchanged)

### Implementation Strategy

1. **Update gemspec**: Replace ace-git-diff with ace-git
2. **Update requires**: Change from internal modules to ace-git
3. **Update usages**: Adapt call sites to use ace-git API
4. **Delete files**: Remove internal Git/PR modules
5. **Delete tests**: Remove tests for deleted modules
6. **Verify**: Run full test suite

## Scope of Work

### Deliverables

#### Delete (Internal Git/PR Components)

- `lib/ace/context/atoms/git_extractor.rb`
  - Purpose: Git operations (diff, log, branch info)
  - Replacement: `Ace::Git::Organisms::DiffOrchestrator`, `Ace::Git::Molecules::BranchReader`

- `lib/ace/context/atoms/pr_identifier_parser.rb`
  - Purpose: Parse PR identifiers (number, owner/repo#number, URL)
  - Replacement: `Ace::Git::Atoms::PrIdentifierParser`

- `lib/ace/context/molecules/gh_pr_executor.rb`
  - Purpose: Execute gh CLI commands to fetch PR diffs
  - Replacement: `Ace::Git::Molecules::PrMetadataFetcher`

- `test/atoms/git_extractor_test.rb`
  - Reason: Tests for deleted git_extractor.rb

- `test/atoms/pr_identifier_parser_test.rb`
  - Reason: Tests for deleted pr_identifier_parser.rb

- `test/molecules/gh_pr_executor_test.rb`
  - Reason: Tests for deleted gh_pr_executor.rb

#### Modify

- `ace-context.gemspec`
  - Changes: Replace `ace-git-diff` with `ace-git`
  - Impact: Different dependency

- `lib/ace/context/organisms/context_loader.rb`
  - Changes: Update `Atoms::GitExtractor` calls to use `Ace::Git` components
  - Integration: Use `Ace::Git::Organisms::DiffOrchestrator` for diffs

- `lib/ace/context/organisms/pr_context_loader.rb`
  - Changes: Replace `Molecules::GhPrExecutor` with `Ace::Git::Molecules::PrMetadataFetcher`
  - Changes: Replace `Atoms::PrIdentifierParser` with `Ace::Git::Atoms::PrIdentifierParser`

#### Keep (Context-Specific Logic)

## API Mapping

| ace-context (current) | ace-git (replacement) |
|----------------------|----------------------|
| `Atoms::GitExtractor.extract_diff(range)` | `Ace::Git::Organisms::DiffOrchestrator.generate(ranges: [range])` |
| `Atoms::GitExtractor.git_diff(range)` | `Ace::Git::Organisms::DiffOrchestrator.generate(ranges: [range]).content` |
| `Atoms::GitExtractor.staged_diff` | `Ace::Git::Organisms::DiffOrchestrator.staged.content` |
| `Atoms::GitExtractor.working_diff` | `Ace::Git::Organisms::DiffOrchestrator.working.content` |
| `Atoms::GitExtractor.current_branch` | `Ace::Git::Molecules::BranchReader.current_branch` |
| `Atoms::GitExtractor.in_git_repo?` | `Ace::Git::Atoms::CommandExecutor.in_git_repo?` |
| `Atoms::PrIdentifierParser.parse(id)` | `Ace::Git::Atoms::PrIdentifierParser.parse(id)` |
| `Molecules::GhPrExecutor.new(id).fetch_diff` | `Ace::Git::Molecules::PrMetadataFetcher.new(id).fetch_diff` |
| `GhPrExecutor::PrNotFoundError` | `Ace::Git::Molecules::PrMetadataFetcher::PrNotFoundError` |
| `GhPrExecutor::GhAuthenticationError` | `Ace::Git::Molecules::PrMetadataFetcher::AuthenticationError` |
