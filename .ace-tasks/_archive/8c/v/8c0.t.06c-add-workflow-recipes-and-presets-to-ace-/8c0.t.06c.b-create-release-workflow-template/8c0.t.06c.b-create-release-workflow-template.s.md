# Create release workflow template

## Objective

Create a recipe and preset for ace-assign that handles version bump and changelog updates without code changes.

## Scope

Create release-only recipe and preset that focuses on:
1. **Bump** - Update version numbers (semver)
2. **Commit** - Commit the version and changelog changes

- [x] `catalog/recipes/release-only.recipe.yml` - Recipe definition
