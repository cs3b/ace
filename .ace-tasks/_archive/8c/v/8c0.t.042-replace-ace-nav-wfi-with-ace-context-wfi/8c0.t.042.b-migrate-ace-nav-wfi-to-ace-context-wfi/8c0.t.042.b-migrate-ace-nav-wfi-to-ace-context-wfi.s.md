# Migrate ace-nav wfi:// to ace-context wfi://

## Objective

Replace all `ace-nav wfi://` references with `ace-context wfi://` across Claude Code commands, workflow instructions, and documentation.

## Behavioral Specification

### User Experience
- **Before**: Commands use `ace-nav wfi://` which returns file path, requiring agent to read file
- **After**: Commands use `ace-context wfi://` which returns content directly (leveraging auto-format)

### Expected Behavior

```markdown
# Before (.claude/commands/ace/commit.md)
read and run `ace-nav wfi://commit`

# After
read and run `ace-context wfi://commit`
```

### Success Criteria

- [ ] 0 occurrences of `ace-nav wfi://` in `.claude/commands/`
- [ ] 0 occurrences of `ace-nav wfi://` in active workflow files
- [ ] Documentation uses `ace-context wfi://` examples
- [ ] `ace-nav wfi://` still works (backwards compatibility preserved)

## Scope of Work

### File Counts (actual)

| Category | Count | Location |
|----------|-------|----------|
| Claude commands | 50 | `.claude/commands/**/*.md` |

| Documentation | ~8 | CLAUDE.md, README.md, docs/*.md |
| Templates | ~2 | ace-integration-claude templates |
| **Total** | ~96 | |

### Exclusions (preserve history)

Do NOT modify:
- `.ace-taskflow/**/done/**` - completed tasks
- `.ace-taskflow/**/retros/**` - retrospectives
- `.ace-taskflow/**/ideas/_archive/**` - archived ideas
- `docs/decisions/**` - ADRs
- `CHANGELOG.md` - historical record

### Pattern Replacement

```
ace-nav wfi://  →  ace-context wfi://
```

Variations to handle:
- `` `ace-nav wfi://name` `` (backtick-wrapped)
- `ace-nav wfi://name` (plain)
- In code blocks and inline

## Technical Approach

### Phase 1: Discovery
```bash
# Find all files needing update
grep -rl "ace-nav wfi://" \
  .claude/commands/ \
  ace-*/handbook/workflow-instructions/ \
  docs/ \
  CLAUDE.md README.md \
  ace-integration-claude/integrations/ \
  --include="*.md" | \
  grep -v "/done/" | \
  grep -v "/retros/" | \
  grep -v "/_archive/" | \
  grep -v "decisions/"
```

### Phase 2: Replace
```bash
# Using sed for batch replacement
find .claude/commands ace-*/handbook/workflow-instructions docs \
  -name "*.md" -type f | \
  xargs grep -l "ace-nav wfi://" | \
  xargs sed -i '' 's/ace-nav wfi:\/\//ace-context wfi:\/\//g'

# Handle root files
sed -i '' 's/ace-nav wfi:\/\//ace-context wfi:\/\//g' CLAUDE.md README.md
```

### Phase 3: Validation
```bash
# Should return 0
grep -r "ace-nav wfi://" .claude/commands/ | wc -l
grep -r "ace-nav wfi://" ace-*/handbook/workflow-instructions/ | wc -l

# Backwards compat check
ace-nav wfi://commit  # Should still return path
```

## Risk Assessment

### Technical Risks
- **Risk**: Some workflows intentionally parse ace-nav path output
  - **Probability**: Low
  - **Impact**: Medium
  - **Mitigation**: Review grep results before bulk replace
  - **Detection**: Search for patterns like `$(ace-nav` or `\`ace-nav ... | ...`

### Integration Risks
- **Risk**: Cached command files in Claude Code
  - **Probability**: Low
  - **Impact**: Low
  - **Mitigation**: Commands are reloaded per session

## Out of Scope

- Deprecating ace-nav tool
- Changing other protocols (guide://, prompt://, tmpl://)
