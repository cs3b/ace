# Rename ace-timestamp to ace-support-timestamp

## Objective

Rename the `ace-timestamp` gem to `ace-support-timestamp` to follow the established `ace-support-*` naming pattern. Keep the executable as `ace-timestamp` for backwards compatibility.

## Scope of Work

### Directory Rename
- `ace-timestamp/` → `ace-support-timestamp/`

### Namespace Change
- `Ace::Timestamp` → `Ace::Support::Timestamp`
- Version bump: 0.1.1 → 1.0.0 (major due to breaking namespace change)

### Executable
- Keep as `ace-timestamp` (no change to exe name)

### lib/ Structure Change
```
ace-timestamp/lib/ace/timestamp.rb       → ace-support-timestamp/lib/ace/support/timestamp.rb
ace-timestamp/lib/ace/timestamp/         → ace-support-timestamp/lib/ace/support/timestamp/
  version.rb                               version.rb
  atoms/                                   atoms/
  cli.rb                                   cli.rb
  commands/                                commands/
```
