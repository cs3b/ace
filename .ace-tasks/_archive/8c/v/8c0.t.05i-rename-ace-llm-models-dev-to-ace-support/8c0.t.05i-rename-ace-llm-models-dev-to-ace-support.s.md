# Rename ace-llm-models-dev to ace-support-models

## Behavioral Specification

### User Experience

- **Input**: Developers and agents use `ace-models` command (was `ace-llm-models`) for model validation, cost tracking, and pricing information
- **Process**: All existing functionality works identically - cache sync, provider listing, model search, pricing
- **Output**: Same output formats with updated gem references

### Expected Behavior

The rename should be transparent to users after migration:

1. **CLI Tool**: `ace-models` replaces `ace-llm-models` with identical behavior
   - `ace-models cache sync` - sync from models.dev
   - `ace-models providers list` - list available providers
   - `ace-models models search` - search models
   - `ace-models models price` - get pricing info
   - All flags and options work identically

2. **Ruby API**: `Ace::Support::Models` module replaces `Ace::LLM::ModelsDev`
   - `require 'ace/support/models'` instead of `require 'ace/llm/models/dev'`
   - All public interfaces unchanged

3. **Configuration**: `.ace/models/` replaces `.ace/llm-models-dev/` (if any)
   - Preset files follow same structure
   - `.ace-defaults/models/` for gem defaults

### Interface Contract

```bash
# CLI Interface (unchanged behavior, new name)
ace-models cache sync              # Sync from models.dev API
ace-models cache status            # View cache status
ace-models providers list          # List all providers
ace-models models search "gpt"     # Search models
ace-models models price "gpt-4o"   # Get pricing

# Ruby API (unchanged interface, new namespace)
require 'ace/support/models'

client = Ace::Support::Models::Client.new
models = client.search("claude")
```

**Error Handling:**
- API errors: Same error messages with new gem name reference
- Cache errors: Same behavior

## Objective

Rename `ace-llm-models-dev` to `ace-support-models` to:
1. Follow the `ace-support-*` naming pattern for infrastructure gems
2. Remove coupling to data source (models.dev) in the name
3. Clarify that this is a support gem for model metadata/pricing
