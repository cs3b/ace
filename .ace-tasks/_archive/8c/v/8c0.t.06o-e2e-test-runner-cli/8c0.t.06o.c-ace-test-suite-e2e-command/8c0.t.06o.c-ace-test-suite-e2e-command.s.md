# ace-test-suite-e2e Command

## Objective

Add `ace-test-suite-e2e` CLI command for running all E2E tests across packages with parallel execution support.

## Behavioral Specification

### User Experience

- **Input**: Optional flags for parallelism and filtering
- **Process**:
  1. Discover all *.mt.md files across packages
  2. Execute tests (sequential or parallel)
  3. Aggregate results
  4. Write comprehensive report
- **Output**: Suite-level summary with per-package breakdowns

### Interface Contract

```bash
# Run full suite (sequential)
ace-test-suite-e2e
# Output:
# Discovered 31 E2E tests across 10 packages
# Running sequentially...
# [1/31] ace-lint/MT-LINT-001: PASS (45s)
# ...
# Summary: 29/31 passed, 2 failed

# Run full suite (parallel)
ace-test-suite-e2e --parallel 4
# Output:
# Running 4 tests in parallel...
# Progress: 31/31 complete
# Summary: 29/31 passed

# Run only affected packages
ace-test-suite-e2e --affected
# Output:
# Analyzing git diff for affected packages...
# Affected: ace-lint, ace-review
# Running 8 E2E tests...

# With CLI args
ace-test-suite-e2e --cli-args dangerously-skip-permissions
```

### Success Criteria

- [ ] Discovers tests across all packages
- [ ] Sequential execution works
- [ ] Parallel execution with configurable concurrency
- [ ] `--affected` flag filters to changed packages
- [ ] Aggregated reporting

  ```ruby
  def run(options: {})
    tests = discover_all_tests
    tests = filter_affected(tests) if options[:affected]

    if options[:parallel]
      run_parallel(tests, options[:parallel])
    else
      run_sequential(tests)
    end
  end

  def run_parallel(tests, max_parallel)
    # Use Open3.popen3 subprocess approach
    # Similar to ace-test-suite ProcessMonitor
    queue = tests.dup
    running = {}
    results = []

    while queue.any? || running.any?
      # Start new processes up to max_parallel
      while running.size < max_parallel && queue.any?
        test = queue.shift
        running[test.id] = spawn_test(test)
      end

      # Check for completed processes
      running.each do |id, process|
        if process.finished?
          results << process.result
          running.delete(id)
        end
      end

      sleep 0.1
    end

    results
  end
  ```
