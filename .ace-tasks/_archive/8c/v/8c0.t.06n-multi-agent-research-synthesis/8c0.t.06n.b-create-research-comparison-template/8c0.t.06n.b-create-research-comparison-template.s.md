# Create Research Comparison Template

## Behavioral Specification

### User Experience
- **Input**: User invokes `ace-bundle tmpl://research-comparison`
- **Process**: System loads structured comparison template
- **Output**: Template for comparing outputs across multiple agents

### Expected Behavior

The template provides a structured format for creating comparison matrices across agent outputs. It includes sections for artifact inventory, quality ratings, conflict tracking, and synthesis decisions with rationale.

### Success Criteria

- [ ] Template created at `ace-handbook/handbook/templates/research-comparison.template.md`
- [ ] Accessible via `ace-bundle tmpl://research-comparison`
- [ ] Includes overview section (task, agents, timestamp)
- [ ] Provides artifact inventory matrix (artifact × agent)
- [ ] Includes quality rating scale (1-5)
- [ ] Has comparison sections by artifact type
- [ ] Contains conflict log with resolution format
- [ ] Documents synthesis decisions with rationale

## Objective

Create template for structured comparison matrix across agent outputs.

## Scope of Work

- Create template following `.template.md` conventions
- Design artifact inventory matrix structure
- Define quality rating scale
- Create conflict log format
- Add synthesis decision documentation format

#### Create

## Implementation Plan

### Execution Steps

- [ ] Step 1: Create template header with metadata placeholders
