# Extract Organisms to ace-config

## Objective

Move orchestration organisms from ace-support-core to ace-config with generic API.

## Files to Extract

### 1. ConfigResolver

**Source:** `ace-support-core/lib/ace/core/organisms/config_resolver.rb`
**Target:** `ace-config/lib/ace/config/organisms/config_resolver.rb`

**Key refactoring:**
```ruby
# Before
def initialize(search_paths: nil, file_patterns: nil, merge_strategy: :replace)

# After
def initialize(
  config_dir: ".ace",
  defaults_dir: ".ace-defaults",
  gem_path: nil,
  project_markers: nil,
  search_paths: nil,
  file_patterns: nil,
  merge_strategy: :replace
)
  @config_dir = config_dir
  @defaults_dir = defaults_dir
  @gem_path = gem_path
  @project_markers = project_markers
  # Pass to ConfigFinder, DirectoryTraverser
end
```

**Add gem_path to cascade:**
```ruby
def build_cascade
  # Existing cascade from DirectoryTraverser
  paths = traverser.find_config_directories

  # Add home
  paths << File.expand_path("~/#{@config_dir}")

  # Add gem defaults (lowest priority)
  if @gem_path
    gem_defaults = File.join(@gem_path, @defaults_dir)
    paths << gem_defaults if Dir.exist?(gem_defaults)
  end

  paths
end
```
