# ace-prompt - Centralize Cache Paths in Config

## Objective

Move hardcoded cache directories and paths from code to `.ace-defaults/prompt/config.yml`.

## Current Issues

Hardcoded paths in multiple files:
- `molecules/prompt_reader.rb` - DEFAULT_PROMPT_PATH=".cache/ace-prompt/prompts/the-prompt.md"
- `molecules/enhancement_tracker.rb` - CACHE_DIR=".cache/ace-prompt/enhance-cache", ARCHIVE_DIR
- `molecules/prompt_archiver.rb` - DEFAULT_ARCHIVE_DIR, PREVIOUS_SYMLINK
- `prompt.rb:14-18` - DEFAULT_MODEL="glite", TEMPERATURE_MIN/MAX

## Scope of Work

### Files to Modify

- `ace-prompt/lib/ace/prompt/molecules/prompt_reader.rb`
- `ace-prompt/lib/ace/prompt/molecules/enhancement_tracker.rb`
- `ace-prompt/lib/ace/prompt/molecules/prompt_archiver.rb`
