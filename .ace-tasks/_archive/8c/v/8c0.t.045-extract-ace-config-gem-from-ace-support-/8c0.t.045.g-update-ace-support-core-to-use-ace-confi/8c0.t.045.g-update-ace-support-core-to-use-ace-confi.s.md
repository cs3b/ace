# Update ace-support-core to Use ace-config

## Objective

Make ace-support-core depend on ace-config and re-export classes for backward compatibility.

## Gemspec Update

```ruby
# ace-support-core/ace-support-core.gemspec
spec.add_dependency "ace-config", "~> 1.0"
```

## Main Entry Point Update

**File:** `ace-support-core/lib/ace/core.rb`

```ruby
require "ace/config"

module Ace
  module Core
    # Re-export ace-config classes for backward compatibility
    module Atoms
      DeepMerger = ::Ace::Config::Atoms::DeepMerger
      YamlParser = ::Ace::Config::Atoms::YamlParser
      PathExpander = ::Ace::Config::Atoms::PathExpander
    end

    module Molecules
      ConfigFinder = ::Ace::Config::Molecules::ConfigFinder
      DirectoryTraverser = ::Ace::Config::Molecules::DirectoryTraverser
      ProjectRootFinder = ::Ace::Config::Molecules::ProjectRootFinder
      YamlLoader = ::Ace::Config::Molecules::YamlLoader
    end

    module Organisms
      ConfigResolver = ::Ace::Config::Organisms::ConfigResolver
      VirtualConfigResolver = ::Ace::Config::Organisms::VirtualConfigResolver
    end

    module Models
      CascadePath = ::Ace::Config::Models::CascadePath
      Config = ::Ace::Config::Models::Config
    end

    # Re-export errors
    ConfigNotFoundError = ::Ace::Config::ConfigNotFoundError
    YamlParseError = ::Ace::Config::YamlParseError
    PathError = ::Ace::Config::PathError
    MergeStrategyError = ::Ace::Config::MergeStrategyError
  end
end
```

## API Wrapper for ace-specific defaults

```ruby
# ace-support-core/lib/ace/core.rb (continued)
module Ace
  module Core
    class << self
      def get(namespace_or_keys, *keys, file: nil, gem_path: nil)
        # Try new .ace-defaults first, fallback to .ace.example during migration
        defaults_dir = if gem_path && Dir.exist?(File.join(gem_path, ".ace-defaults"))
                         ".ace-defaults"
                       elsif gem_path && Dir.exist?(File.join(gem_path, ".ace.example"))
                         ".ace.example"  # Migration fallback
                       else
                         ".ace-defaults"
                       end

        resolver = ::Ace::Config.create(
          config_dir: ".ace",
          defaults_dir: defaults_dir,
          gem_path: gem_path
        )
        # ... existing logic with ace-specific defaults
      end
    end
  end
end
```

**Migration Note**: The fallback to `.ace-defaults` ensures backward compatibility during the transition period (157.08). Remove fallback after all gems are migrated.
