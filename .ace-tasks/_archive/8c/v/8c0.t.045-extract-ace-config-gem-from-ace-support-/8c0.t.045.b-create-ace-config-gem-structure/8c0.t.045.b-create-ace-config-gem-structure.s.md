# Create ace-config Gem Structure

## Objective

Set up the gem skeleton following ace-* conventions with ATOM architecture.

## Directory Structure

```

├── bin/

├── lib/
│   └── ace/
│       ├── config.rb            # Main entry point
│       └── config/

│           ├── atoms/
│           │   ├── deep_merger.rb

```

## Gemspec Requirements

```ruby
Gem::Specification.new do |spec|
  spec.name = "ace-config"
  spec.summary = "Generic configuration cascade management"
  spec.description = "Reusable configuration cascade with customizable folder names"

  # No runtime dependencies - stdlib only
  spec.add_development_dependency "minitest", "~> 5.0"
  spec.add_development_dependency "rake", "~> 13.0"

  spec.required_ruby_version = ">= 3.2.0"

  spec.files = Dir.glob("{lib,test,.ace-defaults}/**/*") + %w[README.md CHANGELOG.md]
end
```
