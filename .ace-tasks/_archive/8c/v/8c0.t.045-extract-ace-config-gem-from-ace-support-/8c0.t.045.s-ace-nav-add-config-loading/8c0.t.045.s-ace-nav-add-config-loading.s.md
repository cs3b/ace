# ace-nav - Add Config Loading

## Objective

Add config loading pattern to ace-nav. Currently `.ace-defaults/nav/config.yml` exists but the gem doesn't load it.

## Current Issues

- `.ace-defaults/nav/config.yml` has configuration options (cache, fuzzy, output settings)
- Main `lib/ace/nav.rb` has no config loading code
- No `reset_config!` method for test isolation

## Scope of Work

### Files to Modify

- `ace-nav/lib/ace/nav.rb` - Add config loading pattern and reset_config!

## Implementation Plan

### Execution Steps

- [x] Add config loading to ace-nav.rb following ADR-022 pattern

  > Use resolve_namespace("nav")
  > Add load_gem_defaults fallback

- [x] Add `reset_config!` method for test isolation

- [x] Verify config values are accessible
  > cache.enabled, cache.directory, cache.ttl
