## Reviewer Entity

```ruby
module Ace::Review::Models
  class Reviewer
    attr_reader :name, :model, :focus, :system_prompt_additions,
                :file_patterns, :weight, :critical

    def initialize(config)
      @name = config[:name] || config["name"]
      @model = config[:model] || config["model"]
      @focus = config[:focus] || config["focus"]
      @system_prompt_additions = config[:system_prompt_additions] || config["system_prompt_additions"]
      @file_patterns = config[:file_patterns] || config["file_patterns"] || {}
      @weight = config[:weight] || config["weight"] || 1.0
      @critical = config[:critical] || config["critical"] || false
    end

    def filter_subject(subject)
      # Apply file_patterns.include/exclude to subject
    end

    def enhance_system_prompt(base_prompt)
      return base_prompt unless system_prompt_additions
      "#{base_prompt}\n\n#{system_prompt_additions}"
    end
  end
end
```

## Configuration

```yaml
reviewers:
  - name: code-quality
    model: google:gemini-2.5-pro
    focus: code_quality
    system_prompt_additions: |
      Focus on code quality, maintainability, and SOLID principles.
      Ignore test files unless they test the code under review.
    file_patterns:
      include: ["lib/**/*.rb", "src/**/*.ts"]
      exclude: ["**/*_test.rb", "**/*.spec.ts"]

  - name: test-coverage
    model: claude:sonnet
    focus: test_quality
    weight: 0.8
    system_prompt_additions: |
      Focus on test coverage, edge cases, and test quality.

  - name: security
    model: openai:gpt-4o
    focus: security
    critical: true  # Security findings are always highlighted
    system_prompt_additions: |
      Focus on security vulnerabilities, OWASP top 10.
```

## Backward Compatibility

When preset uses legacy `model:` or `models:` format:
```ruby
# Legacy: model: "google:gemini-2.5-flash"
# Converted to:
Reviewer.new(name: "default", model: "google:gemini-2.5-flash")

# Legacy: models: ["google:gemini-2.5-flash", "openai:gpt-4o"]
# Converted to:
[
  Reviewer.new(name: "reviewer-1", model: "google:gemini-2.5-flash"),
  Reviewer.new(name: "reviewer-2", model: "openai:gpt-4o")
]
```
