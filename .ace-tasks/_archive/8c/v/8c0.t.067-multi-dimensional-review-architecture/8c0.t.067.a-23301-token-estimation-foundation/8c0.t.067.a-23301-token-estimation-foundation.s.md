# 233.01: Token Estimation Foundation

## Goal

Reliable token counting and model context limit resolution.

## Deliverables

1. `ace-review/lib/ace/review/atoms/token_estimator.rb` - chars/4 heuristic
2. `ace-review/lib/ace/review/atoms/context_limit_resolver.rb` - model → context limit
3. Unit tests for both atoms

## Interface

```ruby
# Token estimation (chars/4 heuristic)
Ace::Review::Atoms::TokenEstimator.estimate("some text")
# → 2 (tokens)

Ace::Review::Atoms::TokenEstimator.estimate_file("/path/to/file")
# → 1500 (tokens)

# Context limit resolution
Ace::Review::Atoms::ContextLimitResolver.resolve("google:gemini-2.5-pro")
# → 2_000_000

Ace::Review::Atoms::ContextLimitResolver.resolve("anthropic:claude-sonnet")
# → 200_000

Ace::Review::Atoms::ContextLimitResolver.default_limit
# → 128_000
```
