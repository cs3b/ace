# Migrate `ace-test-runner-e2e` Handbook, Workflows, and Templates to New TC Goal-Mode Format

## Scope

Single owner task for documentation and workflow alignment after live migration + runner capability changes.

This task updates all `ace-test-runner-e2e` guides/workflows/templates to match the canonical format:
- Scenario IDs: `TS-<PACKAGE_SHORT>-<NNN>[-slug]`
- Standalone goal-mode files: `TC-*.runner.md`, `TC-*.verify.md`
- Artifact layout: `results/tc/{NN}/`
- Report schema: `tcs-passed`, `tcs-failed`, `tcs-total`, `failed[].tc`
- CLI split: `ace-test-e2e` (single package), `ace-test-e2e-suite` (suite)
