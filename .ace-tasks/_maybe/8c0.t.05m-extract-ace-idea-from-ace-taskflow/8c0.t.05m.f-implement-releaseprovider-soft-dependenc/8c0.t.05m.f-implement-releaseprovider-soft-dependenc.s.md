# Implement ReleaseProvider soft dependency

## Objective

Implement the ReleaseProvider molecule that provides soft dependency on ace-taskflow for release detection. This is the key integration point that allows ace-idea to work standalone or integrated.

## Scope

Create ReleaseProvider that:
- Detects if ace-taskflow is loaded (via `defined?(Ace::Taskflow)`)
- Checks if .ace-taskflow directory exists
- Gets current release from taskflow if available
- Falls back to standalone storage otherwise

## Deliverables

### Create
- `ace-idea/lib/ace/idea/molecules/release_provider.rb`
- `ace-idea/test/molecules/release_provider_test.rb`

### Update
- `ace-idea/lib/ace/idea/organisms/idea_writer.rb` - use ReleaseProvider
- `ace-idea/lib/ace/idea/molecules/idea_loader.rb` - use ReleaseProvider

## Implementation Plan

### Execution Steps

- [ ] Create ReleaseProvider molecule:
  ```ruby
  module Ace::Idea::Molecules
    class ReleaseProvider
      def current_release
        taskflow_release || standalone_release
      end

      def storage_root
        use_taskflow_storage? ? ".ace-taskflow" : config.dig("idea", "root") || ".ace-ideas"
      end

      def ideas_directory
        if use_taskflow_storage?
          File.join(storage_root, current_release, "ideas")
        else
          File.join(storage_root, current_release)
        end
      end

      private

      def use_taskflow_storage?
        taskflow_release.present?
      end

      def taskflow_release
        return nil unless compatible_taskflow?
        return nil unless Dir.exist?(".ace-taskflow")

        resolver = Ace::Taskflow::Molecules::ReleaseResolver.new
        release = resolver.current_release
        release if release && release != "backlog"
      rescue => e
        # Log at debug level for observability
        warn "[ReleaseProvider] taskflow detection failed: #{e.message}" if ENV["DEBUG"]
        nil
      end

      def compatible_taskflow?
        return false unless defined?(Ace::Taskflow)
        return false unless defined?(Ace::Taskflow::VERSION)

        Gem::Version.new(Ace::Taskflow::VERSION) >= Gem::Version.new("0.9.0")
      rescue
        false
      end

      def standalone_release
        config.dig("idea", "default_release") || "inbox"
      end
    end
  end
  ```
- [ ] Add tests for both modes:
  - Test standalone mode (no ace-taskflow)
  - Test integrated mode (ace-taskflow present)
  - Test fallback when taskflow has no release
  - Test version compatibility check (incompatible version returns standalone)
- [ ] Update IdeaWriter to use ReleaseProvider

## Technical Notes

**Behavior matrix:**
| ace-taskflow | .ace-taskflow/ | Release | Storage |
|--------------|----------------|---------|---------|
| Not loaded | - | - | `.ace-ideas/inbox/` |
| Loaded | Missing | - | `.ace-ideas/inbox/` |
| Loaded | Exists | None | `.ace-ideas/inbox/` |
| Loaded | Exists | v.0.9.0 | `.ace-taskflow/v.0.9.0/ideas/` |
