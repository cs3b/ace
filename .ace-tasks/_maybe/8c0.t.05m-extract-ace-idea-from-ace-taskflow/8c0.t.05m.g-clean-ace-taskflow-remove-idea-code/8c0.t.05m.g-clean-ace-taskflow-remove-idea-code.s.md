# Clean ace-taskflow - remove idea code

## Objective

Remove all idea-related code from ace-taskflow after it has been moved to ace-idea. Add optional ace-idea dependency and backward compatibility wrapper.

### Add

**Wrapper command:**
```ruby
# lib/ace/taskflow/cli/idea.rb
module Ace::Taskflow::CLI
  class Idea < Dry::CLI::Command
    desc "Manage ideas (delegates to ace-idea)"

    def call(**options)
      if defined?(Ace::Idea)
        Ace::Idea::CLI.start(ARGV.drop(2))
      else
        warn "ace-idea gem not installed. Run: gem install ace-idea"
        1
      end
    end
  end
end
```

## Technical Notes

**Important:** Do NOT add ace-idea to ace-taskflow.gemspec. RubyGems doesn't support "optional" runtime dependencies. Use `defined?(Ace::Idea)` runtime detection only.

**Files that may still reference ideas (need update):**
- `molecules/stats_formatter.rb` - uses IdeaLoader for counts
- `molecules/integrity_validator.rb` - validates idea structure

**Update pattern:**
```ruby
# Before
idea_count = Ace::Taskflow::Molecules::IdeaLoader.new.count_by_release

# After
if defined?(Ace::Idea)
  idea_count = Ace::Idea::Molecules::IdeaLoader.new.count_by_release
else
  idea_count = {}
end
```
