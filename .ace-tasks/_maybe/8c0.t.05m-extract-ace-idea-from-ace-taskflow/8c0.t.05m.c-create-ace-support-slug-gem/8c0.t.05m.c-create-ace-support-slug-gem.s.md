# Create ace-support-slug gem

## Objective

Consolidate slug generation utilities scattered across multiple gems into a single `ace-support-slug` gem. This provides consistent slug sanitization and generation patterns.

## Scope

Consolidate from multiple locations:
- `ace-taskflow/atoms/slug_sanitizer.rb` → foundational sanitizer
- `ace-review/atoms/slug_generator.rb` → URL-safe slugs
- `ace-git-worktree/atoms/slug_generator.rb` → git branch validation
- `ace-taskflow/molecules/task_slug_generator.rb` → semantic slugs

## Technical Notes

**Sanitizer API:**
```ruby
Ace::Support::Slug::Atoms::Sanitizer.sanitize(text, max_length: 50)
# => "my-clean-slug"
```

**GitBranchBuilder API:**
```ruby
Ace::Support::Slug::Molecules::GitBranchBuilder.from_title("My Feature Branch")
# => "my-feature-branch"
```

**SemanticBuilder API:**
```ruby
Ace::Support::Slug::Molecules::SemanticBuilder.from_task_title("Add OAuth authentication")
# => { type: "feat", component: nil, keywords: "oauth-authentication" }
```

**Note:** LlmSlugGenerator stays in ace-idea (LLM-specific, not shared utility)
