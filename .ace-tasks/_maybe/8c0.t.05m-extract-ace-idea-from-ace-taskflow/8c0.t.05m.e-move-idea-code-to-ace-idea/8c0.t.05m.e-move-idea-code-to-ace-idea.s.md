# Move idea code to ace-idea

## Objective

Move all idea-related code from ace-taskflow to ace-idea gem. This is the largest subtask, involving ~2,500 lines of code.

## Scope

### Files to Move

**Models:**
- `ace-taskflow/lib/ace/taskflow/models/idea.rb` → `ace-idea/lib/ace/idea/models/idea.rb`

**Atoms:**
- `ace-taskflow/lib/ace/taskflow/atoms/id_title_extractor.rb` → `ace-idea/lib/ace/idea/atoms/`

**Molecules:**
- `ace-taskflow/lib/ace/taskflow/molecules/idea_loader.rb` (~420 LOC)
- `ace-taskflow/lib/ace/taskflow/molecules/idea_enhancer.rb` (~290 LOC)
- `ace-taskflow/lib/ace/taskflow/molecules/idea_directory_mover.rb` (~350 LOC)
- `ace-taskflow/lib/ace/taskflow/molecules/idea_structure_validator.rb`
- `ace-taskflow/lib/ace/taskflow/molecules/idea_display_formatter.rb`
- `ace-taskflow/lib/ace/taskflow/molecules/idea_arg_parser.rb`
- `ace-taskflow/lib/ace/taskflow/molecules/file_namer.rb` (idea-specific)
- `ace-taskflow/lib/ace/taskflow/molecules/llm_slug_generator.rb` (LLM-specific)

**Organisms:**
- `ace-taskflow/lib/ace/taskflow/organisms/idea_writer.rb` (~405 LOC)
- `ace-taskflow/lib/ace/taskflow/organisms/idea_scheduler.rb`

**Commands:**
- `ace-taskflow/lib/ace/taskflow/commands/idea_command.rb` (~585 LOC)
- `ace-taskflow/lib/ace/taskflow/commands/ideas_command.rb`
- `ace-taskflow/lib/ace/taskflow/commands/idea/*.rb` (nested subcommands)

**Tests:**
- All corresponding test files
