# ace-scheduler Package

## Objective

Create the `ace-scheduler` gem with full functionality: gem structure, CLI commands, cron parsing, task execution, state management, and cron installation.

## Behavioral Specification

### User Experience

**List Tasks:**
```bash
ace-scheduler list
# Output:
# ## Scheduled Tasks
#
# | Task | Schedule | Next Run | Last Run | Status |
# |------|----------|----------|----------|--------|
# | daily-tests | 0 9 * * * | 2026-02-03 09:00 | 2026-02-02 09:00 | Enabled |
# | weekly-e2e | 0 6 * * 1 | 2026-02-08 06:00 | 2026-02-01 06:00 | Enabled |
#
# ## Event Triggers
#
# | Event | Trigger Count | Description |
# |-------|---------------|-------------|
# | pr-merged | 2 | Post-merge verification |
# | release-started | 2 | Pre-release gates |
```

**Run Task:**
```bash
ace-scheduler run daily-tests
# Output:
# Running task: daily-tests
# Command: ace-test-suite --profile 10
# Status: success (exit code 0)
# Duration: 45s
```

**Emit Event:**
```bash
ace-scheduler emit pr-merged
# Output:
# Emitting event: pr-merged
# Triggers: 2
#   [1/2] ace-test-suite --changed: success (30s)
#   [2/2] ace-e2e-test --affected: success (120s)
# All triggers completed successfully
```

**Status:**
```bash
ace-scheduler status
# Output:
# ## Next Scheduled Runs
# | Task | Next Run | In |
# |------|----------|-----|
# | daily-tests | 2026-02-03 09:00 | 19h 5m |
```

**Cron Management:**
```bash
ace-scheduler cron install
# Output:
# Installing cron jobs...
# Added: 0 9 * * * cd /path/to/project && ace-scheduler run daily-tests
# Cron jobs installed successfully
```

## Security Considerations

1. **Config files are trusted**: Commands defined by project maintainers
2. **No remote config**: Config is local-only
3. **User-level cron**: Runs with user permissions
4. **No command interpolation**: Commands passed directly to shell
5. **Audit trail**: All executions logged to state/history/

## Concurrency Strategy

- StateManager acquires exclusive lock before state operations
- Lock file: `.ace/scheduler/state/.lock`
- Lock timeout: 10 minutes (configurable)
