# Local Task Scheduler

## Objective

Create `ace-scheduler` for local cron and event-driven task automation with LLM-friendly output. This provides local scheduling for maintenance tasks and release automation.

## Behavioral Specification

### User Experience

**List Tasks:**
- **Input**: `ace-scheduler list`
- **Output**: LLM-friendly table of tasks and events

**Run Task:**
- **Input**: `ace-scheduler run daily-tests`
- **Output**: Executes command, shows result

**Emit Event:**
- **Input**: `ace-scheduler emit pr-merged`
- **Output**: Triggers all associated actions

### Interface Contract

```bash
# Scheduler Commands
ace-scheduler list                 # List all tasks and events
ace-scheduler run daily-tests      # Run a task immediately
ace-scheduler emit pr-merged       # Emit event, trigger actions
ace-scheduler status               # Show next scheduled runs
ace-scheduler cron install         # Install to system cron
ace-scheduler cron uninstall       # Remove from system cron
```

### Success Criteria

- [ ] `ace-scheduler` can list, run, and manage scheduled tasks
- [ ] Event-driven triggers work (e.g., `ace-scheduler emit pr-merged`)
