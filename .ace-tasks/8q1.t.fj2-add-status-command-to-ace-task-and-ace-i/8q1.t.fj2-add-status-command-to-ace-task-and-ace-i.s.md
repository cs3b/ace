# Add status command to ace-task and ace-idea

Lightweight per-tool status overview commands:
- `ace-task status` — next 3 tasks + summary stats + 9 recently done (configurable)
- `ace-idea status` — next 7 ideas + summary stats + 7 recently done (configurable)

## Implementation

### ace-support-items (shared infrastructure)
- `RelativeTimeFormatter` atom — formats Time into relative strings (just now, 5m ago, 2h ago, etc.)
- `StatusCategorizer` molecule — splits items into up_next (pending, root-only, sorted by ID) and recently_done (done from all folders, sorted by file mtime desc)

### ace-task
- New `status` CLI command with `--up-next-limit` and `--recently-done-limit` options
- `format_status` and `format_status_line` methods on TaskDisplayFormatter
- Config defaults: `status: { up_next_limit: 3, recently_done_limit: 9 }`

### ace-idea
- New `status` CLI command with same option pattern
- `format_status` and `format_status_line` methods on IdeaDisplayFormatter
- Config defaults: `status: { up_next_limit: 7, recently_done_limit: 7 }`

## Output Format

```
Up Next:
  ○ 8pp.t.q7w  Fix login bug
  ○ 8pp.t.r2x  Add pagination

Tasks: ◇ 2 | ○ 12 | ▶ 3 | ✓ 45 • 63 total

Recently Done:
  ✓ 8pp.t.n1a  Complete API docs       (2h ago)
  ✓ 8pp.t.m2b  Fix broken tests        (1d ago)
```
