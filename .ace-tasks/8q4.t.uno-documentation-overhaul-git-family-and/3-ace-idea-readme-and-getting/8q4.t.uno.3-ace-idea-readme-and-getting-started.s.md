# ace-idea README and Getting Started

## Objective

Rewrite ace-idea's README from a 98-line technical reference into a landing page. The current README contains CLI usage with full flag examples, Ruby API with code samples, frontmatter schema YAML, and internal implementation details (b36ts IDs, folder-based storage). Create a tutorial-style getting-started.md.

## Behavioral Specification

### User Experience
- **Input**: Current ace-idea/README.md (98 lines)
- **Process**: Rewrite README as landing page, create getting-started tutorial, write VHS tape, create usage.md + handbook.md, update gemspec, audit handbook
- **Output**: Rewritten README.md (~60 lines), new docs/getting-started.md, new VHS tape + GIF committed to docs/demo/, new docs/usage.md, docs/handbook.md, updated gemspec

### Expected Behavior

New README should convey: "Capture and manage ideas from anywhere -- clipboard, terminal, or LLM enhancement -- so nothing gets lost."

Key messaging:
- Capture ideas quickly from text, clipboard, or with LLM enhancement
- Organize with special folders (_next, _maybe, _anytime, _archive) for GTD-style flow
- 6-command CLI (create, show, list, update, status, doctor) keeps it simple
- Compact b36ts IDs make references short and sortable

Current README problems to fix:
- CLI usage lists every flag combination (belongs in --help)
- Ruby API code samples (belongs in code docs or getting-started)
- Frontmatter schema YAML (internal detail, not landing page material)
- Implementation details about b36ts IDs and folder structure (belongs in docs)

### Interface Contract

README.md (~60 lines):
- Tagline: "Capture and organize ideas from anywhere -- clipboard, terminal, or LLM"
- Demo GIF
- Why section (value-first, no commands)
- Works With (integration packages)
- Agent Skills (package-owned only)
- Features: clipboard capture, LLM enhancement, GTD-style folders, simple 6-command CLI
- Documentation links to getting-started.md and --help
- Part of ACE footer

docs/getting-started.md:
- Prerequisites (Ruby)
- First idea: `ace-idea create "My idea" --title "Feature X"`
- Capturing from clipboard: `ace-idea create --clipboard`
- LLM enhancement: `ace-idea create --clipboard --llm-enhance`
- Organizing with folders: `ace-idea update q7w --move-to next`
- Listing and filtering: `ace-idea list --in next`
- Common Commands table
- What to try next (update metadata, integrate with tasks)

VHS tape: Show `ace-idea create "Dark mode for night coding" --title "Dark mode"` then `ace-idea list`
