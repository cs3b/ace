# Migrate ace-handbook-integration-codex to the fast/feat/e2e testing model

## Objective

Move ace-handbook-integration-codex into the new `fast`/`feat`/`e2e` model with a package-specific migration shape that matches its current test surface.

## Package Migration Shape

- Fast-only package.
- Move flat or command-only deterministic tests into `test/fast/`.
- Do not create `test/feat/` or `test/e2e/` unless new package-owned behavior later justifies them.

## Expected Behavior

1. Move current deterministic package tests into `test/fast/`.
2. Keep `ace-test <package>` on the fast loop only.
3. Update package-local config and docs to describe the fast-only contract.
4. Do not invent deterministic `feat` or scenario `e2e` coverage for this migration.

## Verification

- `ace-test ace-handbook-integration-codex`
- `ace-test ace-handbook-integration-codex all`

## Success Criteria

- [ ] Current deterministic coverage runs from `test/fast/`.
